# 舵桨与推进器选型

## 1. 推进器类型详解

### 1.1 常规螺旋桨

**固定螺距螺旋桨（FPP）：**

| 参数 | 说明 |
|------|------|
| 螺距 | 固定不可调 |
| 转向 | 通过反转电机实现倒车 |
| 效率 | 设计工况下最高 |
| 适用 | 定速定航线船舶（渡轮、客船） |
| 成本 | 最低 |

**可调螺距螺旋桨（CPP）：**

| 参数 | 说明 |
|------|------|
| 螺距 | 液压/电动调节，0 到最大连续可调 |
| 转向 | 电机恒定转向，通过螺距角实现倒车 |
| 效率 | 非设计工况优于 FPP |
| 适用 | 多工况船舶（拖轮、工程船） |
| 成本 | 高（毂机构复杂） |

### 1.2 全回转舵桨

```
        舵桨一体
    ┌───────────┐
    │  电机/齿轮箱 │ ← 可 360° 旋转
    │    桨毂     │
    │  ╱╲ 螺旋桨 ╱╲│
    └───────────┘
```

**特点：**
- 推进+转向一体，无需独立舵
- 360° 全回转，操纵性极好
- 可水上电机（Z型驱动）或水下电机（POD）
- 内河船舶首选

**选型参数：**
- 桨径：受吃水限制，一般 D ≤ 0.8T
- 盘面比：0.55-0.75（内河取大值防空化）
- 叶数：4 叶（常用）或 3 叶（大桨径）

### 1.3 吊舱推进器（POD）

**特点：**
- 电机安装在吊舱内，直接驱动螺旋桨
- 无齿轮箱（直驱），效率高
- 噪音极低，适合客船和豪华游船
- 360° 转向，操纵性优秀
- 成本最高，维护需要干坞

### 1.4 喷水推进

| 优点 | 缺点 |
|------|------|
| 浅水适应性好 | 低速效率差 |
| 无外露运动部件，安全 | 噪音较大 |
| 高速效率高（> 30 节） | 进水口易堵塞 |
| 倒车通过导流板实现 | 倒车效率低 |

## 2. 选型流程

### 2.1 决策矩阵

```
吃水限制？
├─ 严格（< 1.5m）→ 喷水推进 或 小桨径全回转
├─ 一般（1.5-3m）→ 全回转舵桨（推荐）
└─ 宽松（> 3m）→ 常规螺旋桨 + 舵

操纵性要求？
├─ 高（港口船、渡轮）→ 全回转舵桨 或 POD
├─ 中（内河客船）→ 全回转舵桨
└─ 一般（货船）→ 常规桨 + 舵

航速要求？
├─ 高速（> 25 节）→ 喷水推进
├─ 中速（10-25 节）→ 常规桨/全回转
└─ 低速（< 10 节）→ 大直径低转速桨（直驱）
```

### 2.2 螺旋桨设计参数

**关键参数及影响：**

| 参数 | 影响 | 选型原则 |
|------|------|----------|
| 桨径 D | 推力、效率 | 越大效率越高，受吃水限制 |
| 螺距比 P/D | 推力特性 | 重载取大值，轻载取小值 |
| 盘面比 Ae/Ao | 空化、强度 | 高速取大值防空化 |
| 叶数 Z | 振动、效率 | 3-4叶效率高，5叶振动小 |
| 转速 n | 效率、空化 | 低转速大直径效率最高 |

### 2.3 推进器-电机匹配校核

**匹配条件：**
1. 电机额定功率 ≥ 推进器设计功率 × 1.1
2. 电机额定转速 = 推进器设计转速（直驱）或经齿轮箱匹配
3. 电机扭矩曲线覆盖推进器阻力扭矩曲线
4. 电机最大转速 ≥ 推进器超速试验转速（1.2 倍额定）
