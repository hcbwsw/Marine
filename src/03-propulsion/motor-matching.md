# 推进电机匹配

## 1. 电机类型对比

### 1.1 永磁同步电机（PMSM）——船用主流

| 参数 | 典型值 |
|------|--------|
| 效率 | 93-97% |
| 功率密度 | 2-5 kW/kg |
| 转速范围 | 100-3000 rpm |
| 调速性能 | 优异（变频调速） |
| 过载能力 | 150%-200% 短时 |
| 维护量 | 极低（无电刷） |
| 成本 | 中高（永磁体贵） |

✅ **船用首选**：效率最高、体积小、维护少

### 1.2 异步感应电机（IM）

| 参数 | 典型值 |
|------|--------|
| 效率 | 90-94% |
| 功率密度 | 1-3 kW/kg |
| 转速范围 | 500-3000 rpm |
| 过载能力 | 150%-200% |
| 维护量 | 低（无电刷，有轴承） |
| 成本 | 低 |

⚠️ 效率低于 PMSM，但成本低、可靠性好，适合辅机驱动

### 1.3 直流无刷电机（BLDC）

- 本质上是 PMSM 的一种，控制方式不同
- 适合小功率应用（< 50 kW）
- 船用较少，主要在小型工作艇

## 2. 匹配原则

### 2.1 功率匹配

$$P_{motor,rated} \geq P_{propeller,design} \times 1.1$$

$$P_{motor,peak} \geq P_{propeller,max}$$

- 电机额定功率应比推进器设计功率大 10%
- 电机峰值功率应覆盖推进器最大需求（加速工况）

### 2.2 转速匹配

**直驱方案：**
$$n_{motor,rated} = n_{propeller,design}$$

**齿轮箱减速方案：**
$$n_{motor,rated} = n_{propeller,design} \times i_{gearbox}$$

其中 i_gearbox 为减速比（通常 2:1 到 10:1）

### 2.3 扭矩匹配

推进器阻力扭矩：

$$T_{prop} = \frac{P_{prop}}{2\pi n_{prop}}$$

电机扭矩曲线应满足：

$$T_{motor}(n) \geq T_{prop}(n) \quad \forall n \in [0, n_{max}]$$

📌 **关键检查**：在 0 转速（启动瞬间）电机应能输出足够扭矩克服静摩擦。

### 2.4 逆变器匹配

$$P_{inv,rated} \geq P_{motor,rated} \times 1.15$$

$$I_{inv,max} \geq I_{motor,peak}$$

- 逆变器容量留 15% 裕度
- 逆变器最大电流覆盖电机峰值电流
- 逆变器直流侧电压匹配电池系统电压

## 3. 电机特性曲线

### 3.1 典型 PMSM 工作区

```
扭矩 T
  │
  │──── 恒扭矩区 ────┐
  │                  ╲
  │                   ╲ 恒功率区
  │                    ╲
  │                     ╲
  └──────────────────────转速 n
  0     n_base        n_max
```

| 区域 | 特征 | 控制方式 |
|------|------|----------|
| 恒扭矩区（0-n_base） | 扭矩恒定，功率线性增加 | id=0 控制（最大扭矩/电流比） |
| 恒功率区（n_base-n_max） | 功率恒定，扭矩下降 | 弱磁控制 |

### 3.2 推进特性匹配

螺旋桨吸收功率：$P_{prop} = k \times n^3$

$$T_{prop} = \frac{P}{2\pi n} = \frac{k \times n^3}{2\pi n} = \frac{k}{2\pi} \times n^2$$

📌 **推进扭矩与转速平方成正比**，电机在低速时轻松驱动，高速时扭矩需求急剧增加。

## 4. 选型示例

### 4.1 案例：内河电动客船

**需求：**
- 推进器设计功率：2 × 25 kW（双桨）
- 设计转速：500 rpm（全回转舵桨）
- 峰值功率：2 × 45 kW（加速工况）

**电机选型：**
- 类型：PMSM 永磁同步电机
- 额定功率：2 × 30 kW（留 20% 裕度）
- 额定转速：500 rpm（直驱，无齿轮箱）
- 峰值功率：2 × 50 kW
- 额定电压：600 V DC
- 效率：≥ 94%
- 防护等级：IP56
- 冷却方式：水冷

**逆变器选型：**
- 额定功率：2 × 35 kW
- 最大电流：2 × 120 A
- 直流侧电压：500-700 V
- 效率：≥ 97%
- 控制方式：FOC 矢量控制
