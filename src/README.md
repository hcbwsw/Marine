# 船舶新能源动力系统知识库

> 从规范到实操的工程知识库

## 项目简介

本知识库聚焦**船舶新能源动力系统**领域，覆盖电池动力、混合动力、电推进等方向，旨在为船舶工程师提供结构化的知识路径、计算工具和实战案例参考。

## 目标读者

- 船舶动力系统工程师
- 新能源船舶项目技术人员
- 船舶设计院相关人员
- 船舶专业学生及研究人员

## 知识库结构

| 部分 | 内容 | 定位 |
|------|------|------|
| 基础篇 | 船舶动力概述、新能源船舶趋势 | 入门路径 |
| 电池系统 | 电池技术、选型、BMS、安全 | 核心子系统 |
| 推进系统 | 电推进、舵桨、电机匹配 | 核心子系统 |
| 能量管理 | EMS、续航估算、能耗优化 | 系统集成 |
| 规范标准 | CCS/DNV/IEC 规范解读 | 合规依据 |
| 计算工具 | 电池容量、推进功率、续航计算 | 实操工具 |
| 案例库 | 各类新能源船舶案例 | 实战参考 |
| 参考资料 | 书目、标准、术语、链接 | 索引资源 |
