# 船舶新能源动力系统工程手册

> 从基础认知到工程落地的完整知识手册

## 手册简介

本手册系统覆盖**船舶新能源动力系统**的全生命周期知识——从基础概念、核心子系统、系统设计方法、规范合规，到验证落地与商业化。适用于新能源电力推进船舶（内河船 + 近海船 + 海船）的研发工程师、项目经理和技术决策者。

## 目标读者

- 船舶动力系统研发工程师
- 新能源船舶项目技术负责人
- 船舶设计院相关人员
- 船级社审图工程师
- 船舶专业学生及研究人员
