# 能量管理系统（EMS）

## 1. EMS 核心功能

### 1.1 功能架构

```
┌─────────────────────────────────────────────┐
│                  EMS 上位机                   │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐    │
│  │ 功率分配  │ │ 负载管理  │ │ 能耗分析  │    │
│  └──────────┘ └──────────┘ └──────────┘    │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐    │
│  │ SOC 管理  │ │ 故障保护  │ │ 数据记录  │    │
│  └──────────┘ └──────────┘ └──────────┘    │
├─────────────────────────────────────────────┤
│                功率控制层（下位机）            │
│  ┌────────┐ ┌────────┐ ┌────────┐         │
│  │电池 PCS│ │发电机控│ │负载分配│         │
│  └────────┘ └────────┘ └────────┘         │
└─────────────────────────────────────────────┘
```

### 1.2 功率分配策略

**纯电池模式（电池充足时）：**
- 电池单独供电
- 维持 SOC 在 30-90% 区间
- 根据负载需求动态调整放电功率

**混合模式（电池 + 发电机）：**
- 电池承担基础负载（平稳功率）
- 发电机承担峰值负载（波动功率）
- 发电机运行在最优效率点（70-90% 负荷）
- 峰值时电池补充，低谷时电池充电

**低电量保护模式：**
- SOC < 20% 时启动发电机
- SOC < 10% 时限制推进功率
- SOC < 5% 时仅保留安全负载

### 1.3 负载优先级管理

| 优先级 | 负载类型 | 说明 |
|--------|---------|------|
| P0（最高） | 舵机、消防泵 | 安全相关，不可断电 |
| P1 | 推进系统 | 核心功能，优先保障 |
| P2 | 导航通信 | 运行必需 |
| P3 | 照明、舱底泵 | 基本生活保障 |
| P4 | 空调、厨房 | 舒适性，可卸载 |
| P5（最低） | 非必需设备 | 紧急时首先断开 |

📌 **功率不足时**：EMS 按优先级从低到高依次卸载，确保安全设备供电。

## 2. 控制策略

### 2.1 规则型控制（最常用）

**基于 SOC 的状态机：**

```
SOC > 80%: 纯电池模式
SOC 50-80%: 纯电池模式（正常）
SOC 30-50%: 混合模式（启动发电机辅助）
SOC 20-30%: 强制混合模式（发电机主供 + 电池补电）
SOC < 20%: 紧急模式（限功率，发电机全开）
SOC < 10%: 安全模式（仅安全负载）
```

**优点**：简单可靠，逻辑清晰，调试容易
**缺点**：无法优化全局能效

### 2.2 优化控制

**等效消耗最小化策略（ECMS）：**
- 将电池电量消耗等效为燃油消耗
- 实时计算最优功率分配比例
- 适合混合动力船舶

**动态规划（离线优化）：**
- 基于已知航线和工况，全局优化功率分配
- 理论最优，但需预知工况
- 适合固定航线渡轮

**模型预测控制（MPC）：**
- 基于未来有限时域预测，滚动优化
- 在线实时计算，适应工况变化
- 计算量大，需高性能控制器

### 2.3 能量回收

**减速制动能量回收：**
- 船舶减速时，螺旋桨变为水轮机驱动电机发电
- 回收能量 = 减速动能 × 传动效率 × 电机效率 × 充电效率
- 典型回收率：10-25%（短航线频繁停靠）
- 需要电机支持四象限运行 + 逆变器双向能量流

## 3. 系统架构

### 3.1 硬件架构

| 层级 | 硬件 | 功能 |
|------|------|------|
| 上位机 | 工业计算机/PLC | 策略运算、人机界面 |
| 下位机 | DSP/MCU 控制器 | 功率控制、保护执行 |
| 通信 | CAN/Ethernet | 内部通信 |
| 接口 | Modbus/PROFIBUS | 与船舶系统集成 |
| HMI | 触摸屏 | 操作员界面 |

### 3.2 EMS 与其他系统的接口

```
EMS ←→ BMS：电池状态（SOC/SOH/电压/温度/告警）
EMS ←→ 推进变频器：功率指令、转速反馈、状态
EMS ←→ 发电机控制器：启停指令、功率设定、状态
EMS ←→ 船舶监控(SCADA)：综合数据显示、报警
EMS ←→ 岸电系统：充电连接、充电功率控制
```
