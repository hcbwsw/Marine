

# 第12章 内河船与海船差异化设计策略

> **本章导读**
> 船用电推系统并非"一套设计打天下"——
> 内河船与海船在**运营环境、规范体系、任务剖面、维保条件**
> 四个维度上存在系统性差异，这些差异直接穿透到三大子系统
> （冷却 / 传动 / 控制）的每一个设计参数。
>
> 本章的核心价值在于：
> **提供一张可操作的"参数差异化决策表"**——
> 当研发工程师面对一个新项目时，首先判断船型归属，
> 然后对照本章逐参数调整，而不是从零开始重新设计。
>
> 内河电推船（长江、珠江、京杭运河）是当前中国市场最大增量，
> 海船电推（海洋渔船、短途客渡、海工辅助）是下一阶段重点。
> 理解两者的系统性差异，是快速复用设计的基础。

## 12.1 运营环境差异——设计边界的根源

### 12.1.1 外部热汇条件差异

```
冷却系统设计边界的最大驱动因素——外部热汇温度：

  内河船（长江/珠江/京杭运河）：
    水温范围：
      枯水期（冬季）：5~15°C（长江下游）
      丰水期（夏季）：25~32°C（长江下游 7~8月高温期）
      极端值：28°C（夏季洞庭湖汇入区最高纪录）
    设计取值：T_sea_design = 30°C（内河船通用设计基准）
    比海船设计值（35°C）低 5°C
    → 冷却系统设计裕度更宽松，换热器可小一号
    → 相图绿区更大（K₁降额不如海船明显）
    
    北方内河（黑龙江/松花江）冬季：
      水温可接近 0°C
      → 乙二醇浓度须增加（防冻点 ≤ -30°C）
      → 换热器须校核低温工况（粘度增大 → 压损增大 → 泵工作点偏移）

  海船（沿海 + 国际航线）：
    水温范围：
      渤海冬季：-2~5°C
      南海全年：26~32°C（台湾海峡夏季最高 31°C）
      热带远洋：30~35°C（设计上限）
      极端热带：37~38°C（波斯湾、红海极端）
    设计取值：T_sea_design = 35°C（CCS/DNV 标准值）
    → 冷却系统须按 35°C 设计，裕度较小
    → K₁降额因子在热带航线有实质影响（参见第10章）

  量化设计影响（以同功率 500kW 推进电机为例）：
    内河（T_sea=30°C）：换热器面积 A_hx = 2.1 m²（计算值）
    海船（T_sea=35°C）：换热器面积 A_hx = 2.8 m²（+33%）
    → 换热器尺寸、重量、安装空间均有显著差异
    → 不可将内河船换热器直接用于海船（裕度不足）
```

### 12.1.2 水质差异及其对冷却系统的影响

```
内河水质 vs 海水的关键差异：

  ┌──────────────────┬─────────────────────┬──────────────────────┐
  │ 特性             │ 内河水               │ 海水                 │
  ├──────────────────┼─────────────────────┼──────────────────────┤
  │ 盐度             │ <0.5‰（淡水）        │ 30~35‰（标准海水）   │
  │ 氯离子浓度       │ 极低（<10 mg/L）     │ 高（~19000 mg/L）    │
  │ 悬浮物           │ 高（长江含沙量大）   │ 低（相对清洁）       │
  │ 藻类/生物污堵    │ 夏季较严重           │ 沿岸污损严重，远洋较轻│
  │ 油污染           │ 港区较严重           │ 港区严重，远洋轻微   │
  │ pH              │ 6.5~8.5（中性偏碱）  │ 8.1~8.3（弱碱）      │
  └──────────────────┴─────────────────────┴──────────────────────┘

  设计影响：

  ① 外循环（海水/河水侧）换热器材质：
     海船：高氯离子腐蚀风险
           → 钛合金（最耐腐，成本高）或 铜镍合金（Cu-Ni 70/30）
           → 绝对禁止：碳钢、普通不锈钢304（点蚀严重）
     内河船：低氯离子，主要威胁为悬浮泥沙磨蚀
           → 316L不锈钢即可满足（耐磨+经济）
           → 可酌情使用 304（南方内河低沙工况）

  ② 滤器设计：
     海船：细目滤网（孔径 1mm）防生物污堵
           自清洗滤器（反冲洗功能，减少手动维护）
     内河船（长江/黄河高含沙段）：
           粗目滤网（孔径 2~3mm）不易堵塞
           但须缩短清洗周期（每月 → 每周）
           对于汛期（7~9月含沙量暴增），须增加旁通应急阀

  ③ 外循环泵选型：
     海船外循环泵：耐海水腐蚀（钛合金叶轮/青铜壳体）
     内河船：普通铸铁/316L泵即可，成本降低30~50%
             但须考虑泥沙磨损（适当增大叶轮间隙）
```

### 12.1.3 航行工况与任务剖面差异

```
两类船型的典型任务剖面对比：

  内河船（长江 武汉—上海 客渡/货船，典型单程 1000km）：
  ┌────────────┬──────────┬───────────┬─────────────────────┐
  │ 工况       │ 功率占比  │ 时间占比   │ 特殊约束            │
  ├────────────┼──────────┼───────────┼─────────────────────┤
  │ 进出港低速  │ 20~30%   │ 15%       │ 最低转速操纵性要求    │
  │ 航道巡航   │ 60~75%   │ 65%       │ 限速（内河限速15节）  │
  │ 过闸等待   │ 5~10%    │ 10%       │ 充电窗口（>2h）      │
  │ 顶流/满载  │ 85~100%  │ 10%       │ 上行顶流功率需求大    │
  └────────────┴──────────┴───────────┴─────────────────────┘

  内河特有工况：
    ① 过船闸：停机等待 1~3h → 利用等待时间补充充电
    ② 顶流航行（上行）：流速 2~4m/s（长江三峡峡谷段 5~6m/s）
                        → 顶流功率需求可比下行高出 40~60%
                        → 设计时须确保电机在顶流工况不过热
    ③ 浅水航行（枯水期）：螺旋桨进水角度变化
                        → 效率下降 5~10%（需CFD验证）
    ④ 弯道频繁：方向舵/侧推使用频繁，辅助用电负荷大

  海船（沿海货船/渔船/巡逻艇，典型单航次 200~1000km）：
  ┌────────────┬──────────┬───────────┬─────────────────────┐
  │ 工况       │ 功率占比  │ 时间占比   │ 特殊约束            │
  ├────────────┼──────────┼───────────┼─────────────────────┤
  │ 港内机动   │ 15~25%   │ 10%       │ 港内限速，转向频繁    │
  │ 沿海巡航   │ 70~85%   │ 70%       │ 波浪引起负载波动      │
  │ 恶劣海况应急│ 90~100%  │ 10%       │ 砰击引起功率冲击      │
  │ 锚泊/等待  │ 5~10%    │ 10%       │ 辅机用电             │
  └────────────┴──────────┴───────────┴─────────────────────┘

  海船特有工况：
    ① 波浪载荷：螺旋桨周期性出水（空转）→ 过流冲击
                → 控制系统须有"螺旋桨负载突变"保护逻辑
    ② 抗横摇：系统须在 ±15°（典型）横倾时正常工作
               → 膨胀水箱须有足够浮力空间（斜置时不短液位）
               → 冷却泵须校核斜置工况 NPSH（吸入条件变化）
    ③ 结冰工况（北方渤海冬季）：
               → 外循环海水进管须防冰堵
               → 控制器须在 -25°C 低温下正常启动

  两类任务剖面对冷却系统的关键影响：
    内河：负载相对平稳（70% 时间在经济航速），热管理稳态为主
    海船：负载波动大（波浪），热管理须应对 10~20s 周期的功率冲击
          → 热容设计更重要（增大冷却液总量缓冲热冲击）
```

## 12.2 三大子系统差异化参数对照表

### 12.2.1 冷却子系统差异

```
冷却子系统核心参数差异化决策表：

  ┌──────────────────────┬─────────────────────┬────────────────────────┐
  │ 设计参数             │ 内河船               │ 海船                   │
  ├──────────────────────┼─────────────────────┼────────────────────────┤
  │ 外部冷源温度设计值    │ 30°C（内河设计基准）  │ 35°C（CCS/DNV 标准）   │
  │ 换热器设计裕度        │ 热侧裕度 ≥ 25%       │ 热侧裕度 ≥ 35%         │
  │ 换热器材质（外循环侧）│ 316L 不锈钢          │ 钛合金 / Cu-Ni 70/30   │
  │ 外循环泵材质          │ 铸铁 / 316L          │ 青铜 / 钛合金叶轮       │
  │ 外循环滤器孔径        │ 2~3 mm（防泥沙）     │ 1 mm（防生物污堵）      │
  │ 滤器清洗周期          │ 每周（汛期每日）      │ 每月（配合靠港）         │
  │ 内循环冷却液          │ 50% 乙二醇（防冻-35°C）│ 50% 乙二醇（防冻-35°C）│
  │ 冷却泵冗余            │ 可接受单泵（成本优先）│ 必须主备泵（可用性要求）  │
  │ 膨胀水箱安装          │ 固定水平安装          │ 须考虑±15°横倾         │
  │ 外循环进水方式        │ 船底进水口（较稳定）  │ 船底进水口+海水箱        │
  │ 防冻措施              │ 北方须加热盘管（冬季）│ 北方渤海须防冰           │
  │ 盐雾防护              │ 不需要（淡水）        │ 所有外露件须防盐雾       │
  │ 降额律 K₁ 设计基准    │ T_sea=30°C，K₁满足度更高│ T_sea=35°C，K₁关键约束│
  └──────────────────────┴─────────────────────┴────────────────────────┘

  关键设计决策差异详述：

  决策1：换热器是否需要自清洗功能？
    内河高沙量段（长江汛期）：
      建议配置旁通阀（换热器污堵时旁通继续运行，手动清洗时不停机）
      换热器可拆卸清洗型（宽流道板换，防泥沙积聚）
    内河清洁段（京杭运河 / 珠江）：
      普通板换，定期拆洗，无需自清洗
    海船：
      配置自清洗滤器（反冲洗型，每 24h 自动冲洗一次）
      换热器 + 手动旁通阀（换热器清洗时系统可继续降额运行）

  决策2：冷却泵冗余要求
    内河渡船/货船：
      通常单泵可接受（靠港维修机会多，停机代价相对低）
      但须有人工干预备用方案（临时接驳应急泵）
    内河客船（载客 > 100人）：
      须双泵冗余（参照海船标准，乘客安全要求）
    海船（所有类型）：
      必须双泵冗余，自动切换≤5s
      客渡船/救援船：须独立电源分路供电（双泵不共用一路电）
```

### 12.2.2 传动子系统差异

```
传动子系统核心参数差异化决策表：

  ┌──────────────────────┬─────────────────────────┬───────────────────────┐
  │ 设计参数             │ 内河船                   │ 海船                  │
  ├──────────────────────┼─────────────────────────┼───────────────────────┤
  │ 水密度               │ ρ = 1000 kg/m³（淡水）  │ ρ = 1025 kg/m³（海水）│
  │ 螺旋桨推力计算基准    │ K_T × 1000 × n² × D⁴   │ K_T × 1025 × n² × D⁴ │
  │ 推力变化量           │ 基准                     │ +2.5%（同桨、同转速）  │
  │ 轴系材质（耐腐蚀）    │ 碳钢轴 + 镀层            │ 不锈钢轴/锌阳极保护    │
  │ 水润滑轴承材质        │ 赛龙/橡胶（淡水）         │ 赛龙/橡胶（海水 OK）   │
  │ 电机防护等级          │ IP44~IP56（内河机舱）    │ IP56~IP66（海船机舱）  │
  │ 电机绝缘等级          │ F级（45°C环境）          │ H级（60°C机舱）        │
  │ 螺旋桨材质            │ 铸铜合金（耐磨，防泥沙）  │ 镍铝青铜（耐海水腐蚀） │
  │ 轴系波浪弯曲分析      │ 不要求（内陆水域）        │ 必须（海况）           │
  │ 轴系密封（防进水）    │ 唇形密封（淡水压力低）    │ 机械密封（海水高压）   │
  │ 扭振分析              │ 必须（CCS内河规范）       │ 必须（CCS/DNV海船规范）│
  │ 轴系校中              │ 静态+热态                 │ 静态+热态+波浪弯曲工况 │
  └──────────────────────┴─────────────────────────┴───────────────────────┘

  密度差异对螺旋桨匹配的工程影响（量化）：

    同一套电机+螺旋桨，从内河迁移到海船：
    
    推力增加：F_thrust = K_T × ρ × n² × D⁴
              海水/淡水推力比 = 1025/1000 = 1.025（+2.5%）
              → 对推力有利（海船获得更大推力）
    
    转矩增加：T_prop = K_Q × ρ × n² × D⁵
              海水比淡水大 2.5%
              → 电机额定点偏移：海水工况转矩+2.5%
              → 若电机按内河满载标定，迁移海船后可能超转矩
              → 建议：海船螺旋桨设计点留出 ≥ 5% 转矩裕度
    
    功率变化：P = T × ω，相同转速下功率+2.5%
              → 若电机额定功率按内河满载设计，海船满载时可能过载
    
    工程建议：
      对于内河/海船通用船型（如两用渡船），
      螺旋桨按海水（ρ=1025）设计，内河运行时有少量富余功率
      反之不可（内河设计的桨在海船会轻微欠转矩，导致未能满载）

  轴系密封（关键差异）：
    内河船（水压低，相对静止水域）：
      唇形密封（橡胶唇形圈）：结构简单，成本低
      更换周期：约 2~3 年（内河维护方便）
      缺点：渗漏量略大（约 1~2 滴/min），须定期检查
    
    海船（水压大，船体运动引起轴相对位移）：
      机械密封：弹簧压紧端面密封，密封效果好
      无泄漏要求：机舱清洁，减少海水腐蚀
      更换周期：约 5 年（海船维护窗口少）
      成本：比唇形密封高 3~5 倍
    
    选型建议：
      内河客船/载客船：建议升级机械密封（减少维护，乘客舒适性）
      内河货船：唇形密封 + 半年检查
      所有海船：机械密封（无商量余地）
```

控制子系统核心参数差异化决策表：

  │ 驾驶台接口            │ 简单（小船可触摸屏）   │ 完整报警/显示系统       │
  │ 充电控制接口          │ 必须（内河充电桩）     │ 可选（岸电充电）        │
  │ 能量管理复杂度        │ 高（充电/放电/过闸）   │ 中（主要是航行能耗优化）│
  │ 降额律基准温度        │ K₁基于30°C            │ K₁基于35°C             │
  │ 螺旋桨出水保护        │ 不需要（内河稳定）     │ 必须（波浪砰击）        │
  │ 横倾运行功能          │ 不需要                 │ 须验证±15°正常工作      │
  │ 船载智能系统接口      │ 可选（部分智能航道）   │ 推荐（AIS/ECDIS对接）  │
  └──────────────────────┴───────────────────────┴───────────────────────┘

  内河控制系统特有功能——充电协调控制：

    内河纯电船典型充电场景：
      ① 码头快充（夜间靠港，1~3h）：大电流快充（C/2~1C）
      ② 过闸等待充电（白天运营间隙，0.5~2h）：中速充电（C/4~C/2）
      ③ 应急补电（电量不足时临近充电站）：快充（1C）

    充电控制器功能需求（内河特有）：
      □ 充电协议：GB/T 27930（DC 快充）
      □ 充电预约：与运营调度系统对接（ETA提前通知充电桩）
      □ SOC预测：根据剩余航程和水流条件，预测到港SOC
                 若预测 SOC < 20%，提前减速或联系补电
      □ 充电优化：快充/慢充切换（减少电池寿命损耗）
      □ 充电安全：充电期间推进系统隔离，防止误操作
      □ 接口兼容：不同码头充电桩品牌兼容（接口适配器）

    能量管理策略——内河特有的"流量感知调速"：
      原理：上行顶流时增加功率维持速度，
            下行顺流时减少功率节省电量
      输入：GPS 航速 + 水流速（流速仪或 AIS 水文数据）
      输出：推荐转速设定点（兼顾速度和能耗）
      节能效果：典型节省 8~12%（长江双向运营验证数据）
      实现：Level 3 能量管理策略模块（见6.1节）

  海船控制系统特有功能——螺旋桨出水保护：

    海况诱因：
      Hs=3m（中等浪）时，船舶纵摇±5°，螺旋桨周期性出水
      出水时负载突降 90%（从满载 → 接近空载）
      返水时负载突增（螺旋桨重新入水，冲击载荷）
      周期：约 6~12s（与浪周期一致）
    
    控制保护需求：
      ① 负载突降检测：
         检测条件：转矩下降速率 > 20%/100ms
         响应：立即限制转速（防止飞车）
         限速时间：保持 2~5s，确认入水后恢复
      
      ② 限转速保护（Anti-Racing）：
         出水时转速限制：n_max_out = n_rated × 0.85
         防止空转飞车损坏推力轴承
      
      ③ 平滑重载：
         入水后，转矩恢复速率限制：≤ 30% T_rated / s
         防止突然重载引起轴系冲击
      
      ④ 参数设定：
         出水检测转矩阈值：< 15% T_rated（可调，依船型整定）
         HIL 验证：必须在 HIL 平台模拟出水/入水周期验证
                   （见第7章 HIL 场景10：螺旋桨负载突变）
```

    ├── 客船（载客 > 12人）
    │     → 消防/救生更严格，电池舱 A-60 防火
    ├── 危险品船
    │     → IMDG（危险货物）额外要求，防爆电气

          → 相对宽松，但须满足 CCS 渔船规范

  步骤4：是否需要同时满足外国船级社要求？
    ├── 是（出口船型）

## 12.4 差异化设计实施路径

### 12.4.1 "海船向内河降规"的可行性边界

```
已有海船设计向内河船型移植时的降规策略：

  可以降规的项目（内河可用更低规格）：
    ✓ 电机绝缘等级：H级→F级（节省成本，但须确认机舱温度≤45°C）
    ✓ 换热器材质：钛合金→316L（内河低氯离子腐蚀）
    ✓ 外循环泵：海水泵→淡水泵（材质降级，成本-30%）
    ✓ 盐雾防护等级：96h→48h（节省表面处理成本）
    ✓ 控制柜IP等级：IP56→IP54（内河机舱环境相对宽松）
    ✓ 轴系校中：取消波浪弯曲分析（内河无此需求）
    ✓ 换热器面积：可按T_sea=30°C缩减约15%（节省重量和成本）

  不可降规的项目（内河同样要求）：
    ✗ 扭振分析（内河规范同样强制要求）
    ✗ 轴系校中基本分析（冷态+热态）
    ✗ 热保护三级逻辑（不影响成本，直接复用）
    ✗ 控制器看门狗/故障安全（安全要求不分船型）
    ✗ FMEA分析（合规要求）
    ✗ 螺旋桨精度等级（ISO 484 适用）

  额外增加的内河特有功能（海船没有的）：
    + 充电协调控制（GB/T 27930）
    + 流量感知调速（能量管理）
    + 过闸等待充电窗口管理
    + 内河浅水效应补偿（吃水变化对螺旋桨效率的修正）

  降规带来的成本节省估算（典型500kW内河纯电推进系统）：
  ┌────────────────────┬─────────────┬──────────┬──────────┐
  │ 降规项目            │ 海船配置成本 │ 内河成本  │ 节省金额 │
  ├────────────────────┼─────────────┼──────────┼──────────┤
  │ 换热器（钛→316L）   │ ~12万元     │ ~7万元   │ ~5万元   │
  │ 外循环泵            │ ~3万元      │ ~1.5万元 │ ~1.5万元 │
  │ 电机绝缘（H→F）     │ 约+5%电机价 │ 基准     │ ~3万元   │
  │ 轴系密封（机械→唇形）│ ~2万元     │ ~0.5万元 │ ~1.5万元 │
  │ 电缆（成束→单根阻燃）│ ~5万元     │ ~3万元   │ ~2万元   │
  │ 合计节省            │             │          │ ~13万元  │
  └────────────────────┴─────────────┴──────────┴──────────┘
  （注：以上为估算，实际取决于功率等级和具体型号，仅供参考）
```

### 12.4.2 "内河船型向海船升级"的关键检查清单

```
当内河设计需要升级为海船适用时，必须逐项检查：

  冷却系统升级检查：
    □ 换热器外循环侧材质升级：316L → 钛合金/Cu-Ni
      （确认供应商可提供同尺寸钛合金板换）
    □ 外循环泵升级：淡水泵 → 海水泵（耐腐材质）
    □ 换热器面积复核：按T_sea=35°C重新计算（可能需增大10~15%）
    □ 滤器降孔径：2mm → 1mm（防海洋生物污堵）
    □ 添加自清洗功能或旁通阀
    □ 冷却泵升级为双泵冗余（若原内河方案为单泵）
    □ 膨胀水箱横倾校核（±15°工况液位是否满足）
    □ 外露管路防腐处理（海水盐雾腐蚀）

  传动系统升级检查：
    □ 螺旋桨设计点复核：按ρ=1025重新校核推力/转矩
    □ 螺旋桨材质升级：铸铜合金 → 镍铝青铜（NiAlBronze）
    □ 轴系材质升级：碳钢 → 不锈钢轴（或增加牺牲阳极保护）
    □ 轴系密封升级：唇形密封 → 机械密封
    □ 轴系校中：新增波浪弯曲工况分析
    □ 电机绝缘升级：F级 → H级（若机舱温度>45°C）
    □ 电机防护等级：IP54 → IP56
    □ 扭振分析：确认已覆盖海况下的激励（波浪引起桨转矩脉动）

  控制系统升级检查：
    □ 控制器盐雾试验：确认型式试验证书为96h（非48h）
    □ 控制柜IP等级：IP54 → IP56
    □ 新增螺旋桨出水保护逻辑（Anti-Racing）
    □ 新增横倾运行验证（HIL注入横倾模拟工况）
    □ 通信协议确认：NMEA 2000（若船级社要求）
    □ 报警系统完善（海船对报警等级和显示要求更高）
    □ 去掉或旁路内河特有功能（充电桩接口/GB/T 27930）
    □ 降额律K₁基准温度调整：30°C → 35°C

  规范合规升级检查：
    □ 切换适用规范：CCS内河规范 → CCS海船规范
    □ CMM矩阵：新增海船专有条款（轴系波浪弯曲、盐雾等）
    □ 型式试验证书：确认环境试验为海船级别
    □ 与CCS沟通：船型变更时是否需要重新审图（通常须）
```

### 12.4.3 典型差异化方案组合推荐

```
基于上述分析，推荐以下四种典型方案组合：

  方案A：内河货船（纯电，500kW，长江下游）

## 12.5 内河特殊工况深度分析

### 12.5.1 浅水效应对螺旋桨性能的影响

```
浅水效应（Shallow Water Effect）：
  定义：当水深 h 与螺旋桨直径 D 之比 h/D < 4 时，
        螺旋桨周围流场受底部边界影响，性能发生变化

  内河典型水深（枯水期）：
    长江中下游枯水期：水深 5~8m（最浅段 <3m）
    京杭运河：设计水深 4m（实际枯水期可到 3m）
    珠江三角洲：4~6m（航道较深）
  
  典型螺旋桨直径（500kW内河推进）：约 1.0~1.4m
  
  浅水参数：h/D = 3.0/1.2 ≈ 2.5（浅水效应显著！）

  浅水效应对推力的影响：
    当 h/D < 4：螺旋桨有效推力系数 K_T 增大（增加 5~15%）
                但同时推进效率 η 下降（下降 5~10%）
    净效果：推力略有增加，但功率需求也增加
            → 对续航有轻微不利影响

  工程处理：
    精确分析：使用 CFD（RANS 方法）计算浅水工况敞水特性
              修正 K_T、K_Q 曲线（浅水修正系数）
    简化处理（工程估算）：
              K_T_shallow = K_T_deep × (1 + 0.08 × (4 - h/D) / 4)
              当 h/D ≥ 4 时：无修正
              当 h/D = 2.5 时：K_T 约增加 3%
    
    设计建议：
      内河螺旋桨选型时，应同时提供深水和浅水（枯水期）两个工况的
      推力/功率计算，验证电机在两个工况下均不过载
```

### 12.5.2 顶流工况的热管理挑战

```
长江上行（武汉→重庆）顶流分析：

  上行流速数据（长江）：
    武汉—宜昌段：流速 1.5~3.5 m/s（正常）
    三峡峡谷段：流速 3.5~6.0 m/s（高流速，电船挑战最大）
    宜昌—重庆段：流速 1.5~4.0 m/s

  维持10节对地航速（上行）所需推力计算：
    顺流（10节对地=10节对水）：推进功率 P₁
    顶流（10节对地，流速3m/s，对水=10+3×1.94=15.8节）：
      推力 ∝ 对水航速² → (15.8/10)² = 2.50 倍推力需求
      功率 ∝ 对水航速³ → (15.8/10)³ = 3.96 倍功率需求！
      → 若顺流满额定功率 500kW，
         顶流（流速3m/s）需要约 500×3.96/2.50 ≈ **792kW**（超出额定！）
    
  工程应对策略：
    ① 接受速度降低：顶流减速至7~8节（对地），保持额定功率范围内运行
       → 上行时间延长，但系统安全
    ② 增大电机功率：按顶流工况设计电机（如600kW额定），
       顺流时有富余功率 → 成本增加20%
    ③ 顶流+储能峰值输出：电机额定500kW，
       顶流时超功率运行（550kW，30min），由控制器K₃监控绕组温度
       → 需要精确的热管理计算，确保绕组不超限
  
  热管理挑战：
    顶流时功率增大→铜耗增大（∝I²，I增大30%则铜耗增大69%）
    → 绕组温升急剧上升
    → 冷却需求峰值增大 50~70%
    
    控制策略：
      内河上行工况识别：GPS速度 + 流速仪（或水文AIS数据）
      识别到顶流：预先将冷却泵转速升至100%（而非等温度高了再响应）
      顶流时降额律K₁基于实测冷却液温度动态调整
      若绕组接近130°C：主动限速（宁可减速，不要停机）
```
