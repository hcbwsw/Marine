# 船用电池技术基础

## 1. 电池类型对比

### 1.1 磷酸铁锂（LFP）

**化学体系：** LiFePO₄ 正极，石墨负极

| 参数 | 数值 |
|------|------|
| 标称电压 | 3.2 V |
| 能量密度 | 120-160 Wh/kg |
| 循环寿命 | 3000-6000 次（80% DOD） |
| 工作温度 | -20°C ~ 60°C |
| 热失控温度 | ~270°C |
| 充电倍率 | 0.5C-1C（常规），2C-4C（快充） |

✅ **船用首选**——安全性好、循环寿命长、原材料丰富

### 1.2 三元锂（NMC/NCA）

**化学体系：** Li(NiₓMnᵧCo₁₋ₓ₋ᵧ)O₂ 正极

| 参数 | 数值 |
|------|------|
| 标称电压 | 3.6-3.7 V |
| 能量密度 | 180-250 Wh/kg |
| 循环寿命 | 1500-3000 次 |
| 热失控温度 | ~210°C |
| 充电倍率 | 1C-2C（常规），可达 5C |

⚠️ 热稳定性差于 LFP，船用需更严格的防护措施，适合对重量敏感的船型

### 1.3 钛酸锂（LTO）

**化学体系：** Li₄Ti₅O₁₂ 负极

| 参数 | 数值 |
|------|------|
| 标称电压 | 2.4 V |
| 能量密度 | 50-80 Wh/kg |
| 循环寿命 | 10000-20000 次 |
| 工作温度 | -40°C ~ 60°C |
| 充电倍率 | 10C+ |

💡 适合港口拖轮等频繁充放电场景，但能量密度低限制了续航

### 1.4 综合对比

| 指标 | LFP | NMC | LTO |
|------|-----|-----|-----|
| 安全性 | ★★★★★ | ★★★ | ★★★★★ |
| 能量密度 | ★★★ | ★★★★★ | ★★ |
| 循环寿命 | ★★★★ | ★★★ | ★★★★★ |
| 成本 | ★★★★ | ★★★ | ★★ |
| 船用适配 | **首选** | 谨慎选用 | 特殊场景 |

## 2. 船用电池关键指标

### 2.1 能量密度（Wh/kg）

电池组级别能量密度通常比电芯低 15-25%（因结构件、BMS、冷却等）：

- 电芯能量密度：160 Wh/kg → 模组：140 Wh/kg → 电池包：120 Wh/kg

### 2.2 循环寿命

- 定义：容量衰减至初始容量 80% 时的充放电次数
- 船用要求：≥ 3000 次（约 8-10 年日均 1 次循环）
- 影响因素：DOD（放电深度）、温度、充放电倍率

📌 **DOD 与寿命关系（LFP 为例）：**

| DOD | 预期循环寿命 |
|-----|-------------|
| 100% | ~3000 次 |
| 80% | ~5000 次 |
| 50% | ~8000 次 |

### 2.3 充放电倍率（C）

- 1C = 1 小时充满/放完
- 船用典型：充电 0.5C-2C，放电 1C-3C（峰值）
- 快充需考虑电池温升和电网容量限制

### 2.4 船用认证要求

| 认证项目 | 标准/规范 |
|----------|-----------|
| 电池系统型式认可 | CCS《船舶应用电池动力规范》 |
| 电池安全 | IEC 62619、UN38.3 |
| BMS 功能安全 | IEC 61508 |
| 防爆（如适用） | IEC 60079 |
| EMC | IEC 61000 系列 |
