# 电池选型与配置

## 1. 选型流程

```
船舶工况分析 → 能量需求计算 → 峰值功率校核 → 电池类型选择
     → 串并联配置 → 冗余设计 → 重量空间校核 → 认证确认
```

### 1.1 工况分析

**典型内河客船运营工况：**

| 工况 | 功率占比 | 持续时间占比 | 特点 |
|------|----------|-------------|------|
| 离港加速 | 150-200% | 5-10% | 峰值功率，大倍率放电 |
| 巡航 | 60-80% | 50-70% | 持续运行，效率优先 |
| 减速进港 | 30-50% | 10-15% | 可能量回收制动 |
| 停靠 | 10-20% | 15-25% | 辅机用电，低功率 |

### 1.2 选型决策树

```
续航要求 > 4小时？
├─ 是 → 能量密度优先 → 考虑 NMC（需加强安全防护）
│       └─ 安全要求极高？ → 仍选 LFP，增加电池组重量
└─ 否 → 安全性优先 → LFP（推荐）

频繁充放电（> 4次/天）？
├─ 是 → 循环寿命优先 → LFP 或 LTO
└─ 否 → LFP 即可

低温环境（< -10°C）？
├─ 是 → 考虑 LTO，或 LFP + 加热保温
└─ 否 → LFP 正常使用
```

## 2. 串并联配置

### 2.1 电压等级选择

| 系统电压 | 适用功率范围 | 电池串联数（LFP 3.2V） | 典型应用 |
|----------|-------------|----------------------|----------|
| 48 V | < 50 kW | 15-16S | 小型工作艇 |
| 96 V | 50-150 kW | 30-32S | 小型客船 |
| 311 V | 150-500 kW | 96-97S | 中型客船 |
| 600 V | 500-2000 kW | 187-188S | 大型客船 |
| 690 V | > 1000 kW | 215-216S | 大型船舶 |

📌 **电压选择原则：**
- 功率越大，电压越高（降低电流，减少线缆损耗 I²R）
- 受限于功率器件耐压（常用 1200V IGBT/SiC）
- 安全考虑：直流 > 60V 属于危险电压，需绝缘监测

### 2.2 并联支路设计

**单支路容量 vs 多支路并联：**

| 方案 | 优点 | 缺点 |
|------|------|------|
| 大单包少并联 | BMS 管理简单 | 单点故障影响大 |
| 多支路并联 | 冗余好，可隔离故障支路 | 均流管理复杂 |

📌 **推荐**：N+1 冗余配置，任意一支路故障，系统仍能运行。

### 2.3 配置示例

**案例：600V 系统，960 kWh**

- 电芯：3.2V 280Ah LFP
- 串联：188S → 系统电压 601.6V
- 并联：6P → 总容量 1680Ah
- 总能量：601.6V × 1680Ah = 1010 kWh（额定）
- 可用能量：1010 × 0.8（DOD）= 808 kWh

## 3. 电池舱布置

### 3.1 布置原则

- **重量平衡**：电池组重量大，需考虑船舶纵倾和稳性
- **安全隔离**：电池舱与客舱、机舱之间 A-60 级防火分隔
- **维护通道**：留有检查和维护空间（≥ 600mm 通道）
- **通风散热**：自然通风或强制通风，排风口通向安全区域
- **防水防潮**：IP56 以上防护等级

### 3.2 典型布置方案

| 船型 | 电池舱位置 | 特点 |
|------|-----------|------|
| 客船 | 底舱中部 | 重心低，稳性好 |
| 渡轮 | 底舱两侧 | 利用空间，平衡布置 |
| 工作船 | 机舱独立隔间 | 靠近用电设备，缩短电缆 |

## 4. 规范要求

### 4.1 CCS 主要要求

- 电池系统需通过 CCS 型式认可
- 电池舱耐火分隔：A-60（与客舱/机舱之间）
- BMS 需具备过压、欠压、过流、过温保护
- 电池舱需设温度监测、烟雾探测、灭火系统
- 电池系统需进行热失控试验

### 4.2 选型清单

- [ ] 电池类型确认（LFP/NMC/LTO）
- [ ] 电芯参数（容量、电压、内阻）
- [ ] 串并联配置（S×P）
- [ ] 系统总能量和总电压

- [ ] 船级社型式认可证书

- [ ] BMS 功能满足规范要求
