## 2. 关键条款摘录

### 2.1 IEC 62619 电池安全要求

**电芯级试验：**

| 试验项目 | 条件 | 判定标准 |
|----------|------|----------|
| 过充 | 1.5 倍充电电压 | 不起火、不爆炸 |
| 过放 | 放电至 0V | 不起火、不爆炸、不泄漏 |
| 外部短路 | < 5mΩ | 不起火、不爆炸 |
| 热冲击 | 130°C，30min | 不起火、不爆炸 |
| 挤压 | 13kN | 不起火、不爆炸 |
| 跌落 | 1m 自由跌落 | 不起火、不爆炸 |

**模组级试验：**

| 试验项目 | 条件 | 判定标准 |
|----------|------|----------|
| 热蔓延 | 触发单体热失控 | 不蔓延到相邻模组 |
| 过充保护 | BMS 失效下过充 | 保护装置动作 |
| 短路保护 | 外部短路 | 保护装置动作 |

### 2.2 IEC 60092-501 电力推进

- 电力推进系统应设计为故障安全型
- 推进变频器应具备过载保护（150%，1 分钟）
- 推进电机应具备超速保护（120% 额定转速）
- 电力推进系统应设置冗余（至少双推进器或双电机）
- 推进控制系统应具备手动/自动切换功能

## 3. 标准应用指南

### 3.1 电池系统设计引用标准

```
电池选型 → IEC 62619（安全）+ IEC 62620（性能）
BMS 设计 → IEC 61508（功能安全）
电池舱设计 → IEC 60092-305（蓄电池）+ 船级社规范
充电系统 → IEC 61851（充电）+ IEC 60092-304（变流器）
电缆选型 → IEC 60092-350 + IEC 60092-352
EMC 设计 → IEC 60945 + IEC 61800-3
```
