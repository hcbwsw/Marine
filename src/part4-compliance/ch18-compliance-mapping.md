

# 第17章 船舶规范与合规映射

> **本章导读**
> 合规不是"事后盖章"——它是贯穿整个研发生命周期的设计约束。
> 本章回答三个核心问题：
> **①** 我的项目需要满足哪些规范/标准？（规范清单）
> **②** 规范的每一条款对应我设计中的哪个参数？（合规映射矩阵）
> **③** 如何在项目早期识别合规风险，避免审图返工？（合规策略）
>
> 本章与第11章（ICD接口管理）和第20章（V&V验证体系）深度关联：
> 规范条款 → 设计需求（第9章）→ 接口参数（ICD）→ 验证活动（VCRM）
> 形成完整的合规追踪链路。

## 11.1 船级社规范体系（完整清单）

### 11.1.1 中国船级社（CCS）——内河船 + 中国籍海船首选

| 规范 | 版本 | 适用范围 | 关键章节 |
|-----|------|---------|---------|
| 《钢质海船入级规范》R001 | 2024 | 海船 | 第4篇（轮机）/ 第6篇（电气）|
| 《钢质内河船舶建造规范》 | 2023 | 内河船 | 第3篇（轮机）/ 第5篇（电气）|
| 《船舶应用电池动力规范》 | 2024 | 纯电/混动 | 全篇（纯电推进强制）|
| 《船舶应用混合动力系统指南》 | 2024 | 混动 | 全篇 |
| 《智能船舶规范》 | 2024 | 智能推进 | 第9章（智能能效）|
| 《船舶电力推进装置检验指南》 | 最新版 | 电力推进 | 全篇 |

```
CCS审图流程（标准路径）：

  阶段一：初步设计审查（Preliminary Design Review）
    提交材料：
      □ 系统总体方案说明书（推进型式、功率配置、主要设备清单）
      □ 电气系统图（主配电板、推进变频器、保护配置）
      □ 冷却系统原理图（管路拓扑、泵配置、控制逻辑）
    审查重点：系统架构合规性（冗余要求、保护层次）
    周期：约4~6周（以CCS实际反馈为准）

  阶段二：详细设计审查（Detail Design Review）
    提交材料：
      □ 轴系扭振计算书（含Campbell图，禁速区说明）
      □ 轴系校中计算书（冷态/热态轴承载荷）
      □ 热管理系统设计说明（换热器选型计算、温升分析）
      □ 保护系统设计书（三级热保护阈值、联锁逻辑）
      □ 电缆选型计算书（截面、敷设、压降）
      □ 接地系统设计（单点接地、EMC措施）
    审查重点：计算书合规性，关键参数符合规范限值
    周期：约6~8周

  阶段三：建造中检验（Survey During Construction）
    关键节点：
      □ 电机出厂型式试验（FAT）—— CCS检验员见证
      □ 变频器出厂试验 —— CCS检验员见证
      □ 轴系安装对中 —— 现场检验
      □ 电气安装检查（电缆敷设、接地）—— 现场检验
    
  阶段四：试航检验（Sea Trial Survey）
    CCS检验员见证项目（最低要求）：
      □ 推进系统热平衡试验（额定功率持续运行）
      □ 紧急停车试验
      □ 倒车试验
      □ 续航能力试验（纯电船）
    签发证书：CCS入级证书 + 国籍证书

注意：不同规范版本可能有差异，以CCS最新发布版本为准，
      立项前主动与CCS预沟通，确认适用规范版本。
```

### 11.1.2 DNV（国际海船，挪威船级社）

| 规范编号 | 标题 | 关键章节与要求 |
|---------|------|--------------|
| DNVGL-RU-SHIP Pt.4 Ch.8 | 电气设备 | 变频器、电机、电缆选型；保护配置；型式试验要求 |
| DNVGL-RU-SHIP Pt.4 Ch.2 | 机械设备 | 轴系、推力轴承、联轴器；扭振分析要求 |
| DNVGL-OS-D201 | 电气装置 | 电气安装要求；接地；EMC |
| DNVGL-CG-0339 | 电力推进专项指南 | 电推系统设计指导；热管理；控制冗余 |

```
DNV与CCS的主要差异（对研发方的影响）：

  ① 型式试验要求：
     DNV：对变频器和电机有明确的型式试验矩阵（环境试验+电气试验）
          型式试验证书有效期5年，需定期换证
     CCS：类似要求，但具体试验项目略有不同
     → 建议：与DNV/CCS同步沟通，确认型式试验覆盖范围

  ② 扭振分析提交要求：
     DNV：要求使用认可的扭振分析软件（如Shaftviewer/TLA等）
          计算书需经DNV认可的第三方审核（较严格）
     CCS：审查计算书，对软件工具无强制要求

  ③ 电池系统认证（纯电船）：
     DNV：要求电池系统经DNV认证，有专项电池规范
          DNVGL-RU-SHIP Pt.6 Ch.2（电池动力船舶）
     CCS：《船舶应用电池动力规范》2024版
     → 两者框架类似，但认证细节不同

  ④ DP（动力定位）船舶额外要求：
     DNV DP 2/3级：推进系统冗余要求极高
          推进系统必须能在单一故障后保持DP能力
          需提交FMEA报告（DNV审查）
```

```
多船级社同时认证策略（适用于出口船型）：

  情况：同一船型需同时满足CCS + DNV（或其他组合）
  
  策略：
    ① 以更严格的规范为基准设计（通常DNV稍严于CCS）
       满足更严规范的设计通常可直接通过较宽松规范审查
    
    ② 统一型式试验计划：
       协调CCS和DNV检验员同时见证同一型式试验
       避免重复试验（节省成本和时间）
    
    ③ 文件双语版本：
       计算书/说明书：中文（CCS）+ 英文（DNV/ABS/LR）
       ICD文件：双语版本（见第11章）
    
    ④ 早期预沟通（Pre-approval Meeting）：
       立项后3个月内，同时约见CCS和DNV
       确认双方对系统架构的要求是否有冲突
       提前解决，代价最小
```

### 11.2.3 环境试验标准（完整版）

| 标准编号 | 名称 | 关键试验项目 | 适用组件 |
|---------|------|------------|---------|
| **IEC 60068-2-1** | 低温试验（Test A） | -25°C/16h（海船）；-40°C/16h（北方内河冬季）| 控制器、变频器、传感器 |
| **IEC 60068-2-2** | 干热试验（Test B） | +70°C/16h（机舱高温环境）| 同上 |
| **IEC 60068-2-30** | 湿热循环（Test Db） | 55°C/95%RH，循环56天 | 控制器、线束、接线盒 |
| **IEC 60068-2-52** | 盐雾试验（Kb法）| 96h（海船外露件）；48h（内河机舱）| 外露金属件、连接器 |
| **IEC 60068-2-6** | 正弦振动（Test Fc）| 2~13.2Hz: ±1mm；13.2~100Hz: 7m/s²；持续时间各轴向各90min | 控制器、变频器、传感器 |
| **IEC 60068-2-27** | 冲击试验（Test Ea）| 15g/11ms（半正弦波），3轴各3次正负方向 | 控制器、连接器 |
| **IEC 60068-2-64** | 随机振动（Test Fh）| 按船级社规定谱型 | 海船电子设备 |
| **IEC 60068-2-14** | 温度冲击（Test N）| -25°C ↔ +70°C，循环5次，转换时间≤30s | 控制器PCB |

```
环境试验执行策略：

  ① 型式试验 vs 常规试验：
     型式试验（Type Test）：首次入级审查时做一次，证书5年有效
                            代表该型号产品满足规范要求
     常规试验（Routine Test）：每批次出厂时做（简化项目，如绝缘电阻+功能测试）
  
  ② 试验顺序优化（节省时间/费用）：
     推荐顺序：振动 → 冲击 → 湿热循环 → 盐雾 → 低温 → 干热
     原因：破坏性强的试验（盐雾/冲击）后置，
           先用无损试验（振动）筛查潜在问题
  
  ③ 内河船 vs 海船差异：
     内河船控制器：最低要求IEC 60068等级可略低于海船
                  盐雾试验：48h（vs海船96h）
                  低温：-25°C（北方）/ -10°C（南方）
     海船控制器：按上表全项执行，型式试验证书必须有效

  ④ 试验机构选择：
     CCS认可的第三方试验机构（如：中国计量院/广电计量/SGS等）
     DNV认可的机构（确认认可清单）
     建议提前与船级社确认试验机构资质
```

### 11.2.4 能效与环保标准

| 标准/法规 | 发布机构 | 适用范围 | 关键要求 |
|---------|---------|---------|---------|
| **IMO EEDI/EEXI** | 国际海事组织 | 国际航行海船 | 能效设计指数/现有船能效指数，电推船有利（零直接排放）|
| **IMO CII** | 国际海事组织 | 国际航行海船（5000GT以上）| 碳强度指标年度评级（A~E）|
| **GB/T 39267** | 中国国标 | 内河电动船 | 内河电动船舶能效要求 |
| **GB/T 34826** | 中国国标 | 电动船舶 | 电动船舶电力系统及动力系统技术要求 |
| **GB/T 27930** | 中国国标 | 充电接口 | 直流快充通信协议（内河充电桩接口）|

```
纯电推进船舶能效合规要点：

  EEDI合规路径（国际航行纯电船）：
    纯电推进船舶直接排放为零（无主机碳排放）
    EEDI计算时电力消耗对应的排放因子：
      岸电来源：按当地电网排放因子计算（中国约0.58kgCO₂/kWh）
    
    实践中，国际海事组织对纯电船EEDI计算方法尚在完善中
    → 建议：提前与主管机关（中国交通部/船级社）沟通确认计算方法

  CII年度评级（运营阶段）：
    CII = 实际碳排放（tCO₂）/ 运输工作量（DWT × 海里）
    纯电船：CII主要取决于电力来源的碳强度和航行效率
    策略：优化航线和航速（最优经济航速对应最低CII）
          充电优先使用绿电（光伏/风电岸电）
    
  内河船能效（GB/T 39267）：
    规定最低能效限值（能耗/吨海里）
    电动船舶通常远优于限值，合规无压力
    但需在交付文件中提供能效计算书
```

## 11.3 合规映射矩阵（需求—规范—验证完整追踪）

### 11.3.1 合规映射矩阵的用途与结构

```
合规映射矩阵（Compliance Mapping Matrix，CMM）：
  
  用途：
    ① 确保每条规范强制条款在设计中有对应的满足措施
    ② 为审图提供直接的合规证据索引（"此条款见设计文件第X章"）
    ③ 提前识别合规风险（尚未满足的条款 → 红色预警）
    ④ 作为VCRM的上游输入（规范要求 → 设计需求 → 验证活动）

  与VCRM的关系：
    CMM（规范→需求）+ VCRM（需求→验证）= 完整合规链
    规范条款 ─→ CMM ─→ 设计需求编号 ─→ VCRM ─→ 验证报告
    
  列结构（共10列）：
  ┌────┬───────────────┬──────────────────────────────────────┐
  │列号│ 列名           │ 填写说明                              │
  ├────┼───────────────┼──────────────────────────────────────┤
  │ A  │ 规范/标准编号   │ CCS/DNV/IEC/ISO 编号                │
  │ B  │ 条款编号        │ 如：CCS R001 4-6-1.3               │
  │ C  │ 条款内容摘要    │ 原文关键要求（≤30字）               │
  │ D  │ 是否强制        │ 强制/推荐/信息性                    │
  │ E  │ 适用船型        │ 内河/海船/纯电/混动（下拉）         │
  │ F  │ 满足方式        │ 分析(A)/仿真(S)/检查(I)/试验(T)    │
  │ G  │ 对应设计文件    │ 计算书/说明书编号（超链接）          │
  │ H  │ 对应需求编号    │ REQ-XXX-XXX（与VCRM关联）          │
  │ I  │ 合规状态        │ ✅已满足/⚠️进行中/❌未满足/N/A      │
  │ J  │ 备注/风险说明   │ 未满足项的整改计划或偏差说明        │
  └────┴───────────────┴──────────────────────────────────────┘
```

## 11.4 审图通过策略与常见问题

### 11.4.1 审图前置工作（避免返工的关键）

```
经验法则：审图通过率与前期准备质量正相关
          充分的前期沟通可将审图轮次从3~4轮减少至1~2轮
          每减少一轮审图：节省约4~6周工期

前置工作清单（提交审图前必须完成）：

  ① 与船级社预沟通（Pre-submission Meeting）
     时间：项目立项后3个月内
     议题：
       □ 确认适用规范版本（规范每年更新，以最新版为准）
       □ 确认型式试验要求（哪些组件需要提供型式试验证书）
       □ 确认计算书深度要求（扭振/热管理/轴系校中的格式）
       □ 特殊技术方案预审（如：无编码器控制、创新冷却方案）
     
     输出：会议纪要（双方签字确认），存档备查
     重要性：★★★★★（最高优先级）

  ② 内部合规自查（Internal Compliance Check）
     基于CMM矩阵，逐条核查：
       □ 所有"强制"条款是否均有对应设计文件
       □ 关键计算值是否满足规范限值（留裕度）
       □ 型式试验证书是否在有效期内（5年）
     
     发现❌未满足项 → 立即启动整改，在提交前关闭
     剩余⚠️进行中项 → 附整改计划提交（说明完成时间节点）

  ③ 计算书质量控制
     扭振计算书：
       □ 软件名称和版本号明确
       □ 输入参数来源明确（电机数据/联轴器数据/桨数据）
       □ Campbell图清晰，禁速区标注明确
       □ 动态扭应力验算齐全
     
     热管理计算书：
       □ 损耗计算有依据（测试数据或供应商数据）
       □ ε-NTU换热计算步骤完整
       □ 最恶劣工况明确（T_sea=35°C，污堵30%）
       □ 设计裕度明确（≥10%以上）
     
     轴系校中计算书：
       □ 冷态/热态双工况
       □ 所有轴承载荷为正
       □ 满足IACS UR M68要求
```

### 11.4.2 常见审图不通过原因与应对

```
CCS审图最常见的退审原因（基于工程经验）：

  ① 扭振分析不充分（发生率最高，约占退审原因35%）
     常见问题：
       · 只给出固有频率，未做Campbell图（激励阶次分析）
       · 联轴器刚度使用标称值而非实测范围
       · 禁速区范围计算错误（应为±10%，而非±5%）
       · 未分析电磁激励（6n阶次）的影响
     
     应对：
       · 严格按IACS UR M13格式提交
       · 联轴器刚度提供供应商实测范围（±20%敏感性分析）
       · Campbell图覆盖0~110%额定转速范围
       · 检验员希望看到的：每个激励阶次的干涉点是否都在禁速区外

  ② 热保护逻辑说明不充分（约占25%）
     常见问题：
       · 三级保护阈值有但未说明来源（如何确定155°C触发值）
       · 降额响应时间未经过试验验证（仅有设计值）
       · 传感器三取中投票逻辑未说明
     
     应对：
       · 热保护设计书：每个阈值的工程推导说明（6.3节思路）
       · 附HIL故障注入试验结果（控制器测试报告）
       · 说明传感器故障时的降级策略

  ③ 电缆计算书缺失或不完整（约占20%）
     常见问题：
       · 仅选型未计算（缺少截面验算）
       · 短路电流未校核
       · 电缆敷设温度修正系数未考虑（机舱高温修正）
     
     应对：
       · 按IEC 60092-353逐回路计算，附表格
       · 提供短路电流计算书（变频器额定短路电流）
       · 机舱温度修正系数按CCS规范取值

  ④ 材质/防护等级不合规（约占10%）
     常见问题：
       · 海水侧换热器使用304SS（应使用316L或钛合金）
       · 机舱电气设备防护等级IP54（应≥IP56）
       · 变频器防护等级不满足机舱要求
     
     应对：
       · 采购文件明确规定材质和防护等级要求
       · 供应商提供材质证书（MTR）和防护等级试验报告

  ⑤ 型式试验证书过期或缺失（约占10%）
     常见问题：
       · 5年前的型式试验证书未更新
       · 仅有国内认证，无船级社认可
     
     应对：
       · 项目立项时核查所有关键组件型式试验证书有效期
       · 提前（≥6个月）安排型式试验换证
```

### 11.4.3 内河船特殊合规要点

```
内河船规范与海船的主要合规差异（对研发方的影响）：

  ① 防护等级要求：
     内河船机舱：一般要求IP44~IP55（低于海船IP56要求）
     但：若内河船有开放甲板上的设备，应按海船IP56处理
     → 策略：统一按IP56设计，超出内河要求，但避免海船版本双线开发

  ② 绝缘等级：
     内河船电机：F级绝缘即可（T_ambient ≤ 45°C）
     海船电机：H级绝缘（T_ambient ≤ 60°C，机舱高温）
     → 对于出口内河船型：按海船H级设计

  ③ 扭振分析：
     内河船：CCS内河规范同样要求扭振分析
              但可简化（对于功率<200kW，简化分析可接受）
     → 具体豁免条件与CCS审图组协商确认

  ④ 锚地充电合规（内河特有）：
     内河渡轮频繁充电（日均20次）：
       充电接口认证：GB/T 27930（直流快充）
       充电桩对接：与运营方和充电桩厂家协调接口标准
       安全：充电期间防爆区域划定（锂电池排气口附近）

  ⑤ 消防要求（纯电内河船）：
     CCS《电池动力规范》：
       电池舱须设固定式气体/气溶胶灭火系统
       或：高压细水雾（二者选一）
       电池舱独立结构（防火等级A-60）
     → 与船厂协调结构设计（属于船总体设计但影响电池安装）
```
