# 电池安全与防护

## 1. 热失控机理

### 1.1 热失控阶段

| 阶段 | 温度范围 | 特征 | 可逆性 |
|------|----------|------|--------|
| 安全区 | < 60°C | 正常工作 | ✅ |
| 警戒区 | 60-90°C | SEI 膜开始分解 | ⚠️ 可控 |
| 危险区 | 90-120°C | 隔膜熔融，内短路开始 | ❌ 临界 |
| 热失控 | > 120°C | 正极分解，电解液燃烧，自放热 | ❌ 不可逆 |
| 蔓延 | > 500°C | 火焰/爆炸，邻近电芯连锁热失控 | ❌ 灾难性 |

### 1.2 触发原因

| 原因 | 机制 | 防范 |
|------|------|------|
| 过充 | 锂枝晶穿刺隔膜 | BMS 过压保护 + 硬件保护 |
| 过温 | 环境高温或散热失效 | 温度监测 + 热管理 |
| 外短路 | 水侵入、绝缘失效 | 绝缘监测 + 快速熔断 |
| 内短路 | 制造缺陷、机械损伤 | 电芯筛选 + 机械防护 |
| 挤压/穿刺 | 碰撞事故 | 电池舱结构防护 |
| 过流 | 大倍率放电 | BMS 过流保护 |

### 1.3 蔓延路径

```
单体热失控 → 模组内蔓延（传导+辐射）→ 模组间蔓延 → 电池簇蔓延 → 电池舱失控
```

📌 **防护目标**：单体热失控后，不蔓延到相邻模组（"不蔓延"设计准则）。

## 2. 船用电池安全规范

### 2.1 CCS 规范要点

| 项目 | 要求 |
|------|------|
| 电池舱分隔 | 与客舱/机舱 A-60 级防火分隔 |
| 电池舱通风 | 独立通风，换气 ≥ 20 次/小时 |
| 温度监测 | 每个模组至少 2 个温度传感器 |
| 烟雾探测 | 电池舱内设置感烟探测器 |
| 灭火系统 | 七氟丙烷或水雾灭火 |
| 泄爆设计 | 电池舱设泄压装置 |
| 紧急切断 | 电池舱外设紧急切断按钮 |
| 热失控试验 | 电池系统需通过热失控不蔓延试验 |

### 2.2 DNV 规范要点

- DNV-RU-Ship Pt.3 Ch.2 Sec.3：电池安全
- 热失控要求：单个电芯热失控不蔓延
- 电池舱温度监测密度更高（每 2m²）
- 强制要求电池系统 FMEA 分析
- 需提交热失控仿真报告

### 2.3 IEC 62619 要求

- 电芯级：过充、过放、短路、热冲击、挤压测试
- 模组级：热蔓延测试（单体触发热失控，验证不蔓延）
- 系统级：功能安全评估（IEC 61508）

## 3. 电池舱设计

### 3.1 舱室要求

```
┌──────────────────────────────────────────┐
│            电池舱设计要求                  │
├──────────────────────────────────────────┤
│  1. 独立舱室，A-60 防火分隔               │
│  2. 独立通风系统（排风通向安全区域）        │
│  3. 温度监测 + 烟雾探测                    │
│  4. 自动灭火系统（七氟丙烷/水雾）           │
│  5. 泄爆口（泄压面积 ≥ 舱容×0.05）         │
│  6. 紧急切断按钮（舱外可达）                │
│  7. 绝缘监测装置                          │
│  8. 防积水设计                            │
│  9. 照明（防爆型）                        │
│  10. 电缆穿舱件防火密封                    │
└──────────────────────────────────────────┘
```

### 3.2 灭火系统对比

| 灭火介质 | 适用性 | 优点 | 缺点 |
|----------|--------|------|------|
| 七氟丙烷（FM200） | 推荐 | 灭火快、残留少 | 用量受限，需定期更换 |
| 细水雾 | 推荐 | 环保、持续降温 | 需防积水，电气绝缘风险 |
| CO₂ | 可用 | 成本低 | 窒息风险，需人员撤离 |
| 泡沫 | 不推荐 | — | 电气短路风险 |
| 干粉 | 不推荐 | — | 清理困难，损坏设备 |

📌 **推荐方案**：七氟丙烷主灭 + 细水雾持续降温，防止复燃。

### 3.3 热管理系统

**风冷 vs 液冷：**

| 方案 | 散热能力 | 复杂度 | 维护 | 适用场景 |
|------|----------|--------|------|----------|
| 风冷 | 一般 | 低 | 简单 | 中小功率（< 200 kW） |
| 液冷 | 强 | 高 | 需防漏 | 大功率（> 200 kW） |

**设计要点：**
- 电池组温差控制在 5°C 以内（延长寿命）
- 进出风口/液冷管路合理布置，避免局部热点
- 低温环境需加热功能（电芯 < 0°C 禁止充电）
- 热管理与 BMS 联动，自动调节
