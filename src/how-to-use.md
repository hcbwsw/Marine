# 如何使用本知识库

## 阅读路径

**如果你是初学者：**
1. 先看「基础篇」了解全貌
2. 按「电池系统」→「推进系统」→「能量管理」顺序学习
3. 遇到规范问题查阅「规范标准」
4. 用「计算工具」做实操练习
5. 参考「案例库」理解实际应用

**如果你是工程师（查资料）：**
- 直接通过目录跳转到对应章节
- 「计算工具」章节提供常用计算方法和公式
- 「规范标准」章节提供条款解读和查找索引

**如果你要做方案设计：**
1. 「规范标准」确认合规要求
2. 「电池系统」+「推进系统」选型
3. 「计算工具」做关键参数计算
4. 「案例库」参考同类项目
5. 「能量管理」优化系统效率

## 文档约定

- 📌 **重要结论**：关键设计准则或规范要求
- ⚠️ **注意**：常见误区或风险点
- 💡 **提示**：实操经验或技巧
- 📐 **公式**：关键计算公式
- 🔗 **参考**：引用的规范条款或文献

## 本地运行

```bash
# 安装 mdBook
cargo install mdbook
# 或
brew install mdbook

# 进入项目目录
cd marine-energy-kb

# 本地预览
mdbook serve --open

# 构建静态网站
mdbook build
```
