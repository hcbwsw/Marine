# 如何使用本手册

## 手册定位

这是一本**船舶新能源动力系统工程手册**，覆盖从基础概念到工程落地的完整链路。既是学习教材，也是设计参考，更是合规工具书。

## 七部分结构

| 部分 | 章节 | 回答的问题 |
|------|------|-----------|
| 一、基础认知 | 第1-2章 | 是什么？有哪些形式？怎么选？ |
| 二、核心子系统 | 第3-6章 | 电池/推进/能量/冷却怎么设计？ |
| 三、系统设计方法 | 第7-13章 | 如何用系统工程方法做研发？ |
| 四、规范与合规 | 第14-19章 | 怎么过船级社审图？ |
| 五、验证与落地 | 第20-24章 | 怎么验证？怎么交付？ |
| 六、专项深化 | 第25-29章 | 混动怎么做？怎么商业化？ |
| 七、工具与资源 | 附录A-E | 计算器/案例/参考在哪？ |

## 阅读路径

### 🎓 新手入门（零基础）

```
第1章（概述）→ 第2章（分类选型）→ 第3章（电池）
→ 第4章（电推进）→ 第24章（90天路线，建立全局观）
```

### 🔧 研发工程师（做设计）

```
第二部分（子系统设计）→ 第三部分（设计方法）
→ 第19章（特有问题避坑）→ 第五部分（验证落地）
```

### 📋 项目经理（管项目）

```
第7章（研发框架）→ 第8章（需求工程）→ 第20章（V&V）
→ 第24章（90天路线）→ 第26章（商业化）
```

### ✅ 合规工程师（过审图）

```
第14-17章（规范体系）→ 第17章（合规映射）
→ 第18章（内河vs海船）→ 第19章（特有问题）
```

### 🧮 快速计算（查工具）

直接打开 **附录A 交互计算工具**：电池容量、推进功率、续航里程、效率链，输入参数实时出结果。

## 文档约定

| 标记 | 含义 |
|------|------|
| 📌 | **重要结论**：关键设计准则或规范要求 |
| ⚠️ | **注意**：常见误区或风险点 |
| 💡 | **提示**：实操经验或技巧 |
| 📐 | **公式**：关键计算公式 |
| 🔗 | **参考**：引用的规范条款或文献 |

## 本地运行

```bash
# 安装 mdBook
brew install mdbook

# 进入项目目录
cd marine-energy-kb

# 本地预览（自动开浏览器）
mdbook serve --open

# 局域网访问（手机也能看）
mdbook serve --hostname 0.0.0.0 --port 3000
```
