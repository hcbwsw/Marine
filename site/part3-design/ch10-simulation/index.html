    第10章 多物理域协同仿真体系¶

    适用范围：内河船 + 海船电驱动系统 覆盖工具：MATLAB/Simulink、AMESim、ANSYS、Maxwell、Fluent、HIL平台 内河/海船各50%覆盖

    7.1 仿真体系总体架构¶

    7.1.1 为什么需要多物理域协同仿真¶

    船用电驱动系统是典型的多物理域强耦合系统，各域之间存在双向耦合：

    电磁域 ←──────────────────────────────────────► 热域
  铜耗/铁耗 → 热源                               温升 → 绕组电阻升高（铜耗增大）
  电流纹波 → 附加损耗                             磁钢温升 → 退磁风险（PMSM）

热域 ←──────────────────────────────────────► 冷却域
  热源分布 → 冷却需求                             冷却液温度/流量 → 热边界条件
  温升分布 → 热应力                               泵功耗 → 额外热源

机械域 ←──────────────────────────────────────► 控制域
  轴系固有频率 → 控制带宽上限                      速度环带宽 → 扭振激励风险
  扭振幅值 → 编码器信号噪声                        电流纹波 → 转矩波动 → 扭振激励

电磁域 ←──────────────────────────────────────► 机械域
  转矩波动（谐波）→ 轴系激励                       转速波动 → d/q轴电流耦合
  齿槽转矩 → 低速振动                             负载突变 → 电流冲击

    如果各域单独仿真，将漏掉以下关键耦合效应：

    漏掉的耦合 典型后果 工程损失
    热-电磁耦合 绕组电阻随温升增大8~15%，铜耗估算偏低 换热器选小→过热降额
    控制-机械耦合 速度环带宽设置过高，激励轴系共振 联轴器损坏、振动超标
    冷却-热耦合 冷却响应滞后未考虑，热保护触发过频 推力损失、乘客投诉
    电磁-机械耦合 谐波转矩激励未纳入扭振分析 轴系疲劳失效提前

│  传动：ANSYS Mechanical / Simpack（扭振+校中）                   │

数据流向：
  L1 → L2：高保真结果作为精确参数（换热系数、Kt/Kq曲线）
  L2 → L3：子系统简化模型参数（热阻、刚度、传递函数）
  L3 ← → HIL：控制器代码在实时环境中验证

    7.2.2 模块一：任务剖面模块¶

    % 任务剖面定义（内河客船示例）
% 五工况循环，总时长 5.5h/航次

profile = struct();
profile.segments = {
%  工况名称      功率比   持续时间(s)  海温(°C)  备注
  '靠泊等待',    0.05,    1800,        30,      '发动机怠速';
  '低速出港',    0.30,    1200,        30,      '进出港限速';
  '经济航速巡航', 0.75,   14400,       35,      '主要工况';
  '最大航速冲刺', 1.00,    1800,       35,      '赶点补时';
  '倒车靠泊',    0.85,     300,        35,      '靠码头';
};

% 任务剖面向等效工况转换
P_eq = sqrt(sum([P_i.^2 .* t_i]) / sum(t_i));  % 热等效功率
% 用于换热器选型基准（非峰值功率！）

    海船任务剖面示例（远洋货船）：

    工况段          功率占比    时长占比    海温范围    说明
─────────────────────────────────────────────────────────
港内操纵         15%         5%        20~30°C    拖轮协助、低速
出/进港          35%        10%        20~30°C    狭水道、限速
大洋巡航         80%        70%        5~35°C     主要工况，变海温
恶劣天气减速     50%        10%        5~35°C     波高>5m限速
应急全速          100%       5%        最高设计值  追赶班期/避险

加权等效功率：P_eq = √(0.15²×0.05 + 0.35²×0.10 + 0.80²×0.70
                     + 0.50²×0.10 + 1.00²×0.05) × P_rated
           ≈ 0.756 × P_rated
→ 换热器应按 75.6% 额定功率选型（加30%裕度），而非100%

    7.2.3 模块二：电机电磁-热耦合模型¶

    Cauer 集总参数热网络（六节点）：

    热源分布：
  Q_cu(t) = 3 × I_rms²(t) × R₀ × [1 + α(T_winding - 20°C)]
            ↑ 注意：电阻随温度变化，需迭代求解！
  Q_fe    ≈ K_h×f×B_m^α×V_core + K_e×f²×B_m²×V_core（近似恒定）
  Q_mech  = k_mech × n^1.5（风摩耗，随转速变化）

六节点 Cauer 热网络：

  Q_cu → [R_cu-slot] → T_winding
            ↓
         C_winding × dT_winding/dt = Q_cu - (T_winding-T_slot)/R_cu-slot

  Q_fe → [R_fe-yoke] → T_core
            ↓
         C_core × dT_core/dt = Q_fe - (T_core-T_yoke)/R_fe-yoke

  T_slot → [R_slot-yoke] → T_yoke → [R_yoke-jacket] → T_jacket
            ↓                          ↓
         C_slot × dT_slot/dt        C_yoke × dT_yoke/dt

  T_jacket → [R_jacket-fluid] → T_fluid_in
  T_fluid_out = T_fluid_in + Q_total/(ṁ × Cp)

各热阻参数估算（2000kW电机典型值）：
  R_cu-slot   = 0.05 K/W  （绕组导热脂填充的槽绝缘）
  R_slot-yoke = 0.03 K/W  （槽楔+铁芯接触）
  R_yoke-jacket = 0.02 K/W（铁芯-冷却套接触，含接触热阻）
  R_jacket-fluid = 0.01 K/W（冷却套-冷却液对流，取决于流速）

  对流热阻计算：R_conv = 1/(h × A_cooling)
  h = Nu × λ_fluid / D_h  （Nu = 0.023 Re^0.8 Pr^0.4，Dittus-Boelter）

    迭代求解流程（热-电磁耦合）：

    初始化：T_winding = T_ambient = 25°C

循环迭代（时间步长 Δt = 0.1s）：
  Step 1：根据 T_winding 计算 R_phase(T) = R_20°C × [1+0.00393×(T-20)]
  Step 2：由 FOC 输出 I_dq 计算 Q_cu(t) = 3×I_rms²×R_phase(T)
  Step 3：求解热网络 ODE，得到新的 T_winding, T_core, T_jacket...
  Step 4：更新 Q_cu（铜耗随温升增大约8~12%），回到Step 1
  Step 5：检查收敛：|T_new - T_old| < 0.1°C → 收敛，否则继续

⚠️ 不迭代的后果：100°C温升时铜耗低估约5%，换热器面积偏小，
   实际运行中冷却不足，过热降额频发。

    7.2.4 模块三：冷却回路热网络模型（AMESim/Simulink）¶

    状态方程（一维热网络）：

内循环（乙二醇水溶液，50%质量分数）：
  Cp = 3500 J/(kg·K)，ρ = 1060 kg/m³

  质量流量：ṁ_inner = f(n_pump, ΔP_system)  ← 泵曲线查表

  热平衡（电机冷却套）：
  T_out_motor = T_in_motor + Q_motor / (ṁ_inner × Cp)

  换热器换热量：
  Q_hx = ε × C_min × (T_hot_in - T_cold_in)
  ε = 1 - exp[-NTU×(1-C_r)] / [1 - C_r×exp(-NTU×(1-C_r))]
  NTU = UA / C_min

  外循环（海水）：
  T_sea_out = T_sea_in + Q_hx / (ṁ_outer × Cp_sea)

动态仿真关键参数（2000kW系统示例）：
  内循环流量：40~80 L/min（随泵速变化）
  换热器NTU：3.5（清洁状态），2.4（30%污堵）
  热容C_motor：85 kJ/K（集总参数）
  时间常数τ：τ = C_motor × R_th_total ≈ 85000 × 0.04 ≈ 3400s ≈ 57min

→ 电机热时间常数约57分钟！
  意味着：短时过载（<10min）不会立即超温，
          但持续过载必然超温，且冷却响应有显著滞后。

    7.2.5 模块四：轴系扭振简化模型¶

    集总参数扭振模型（三质量体系统）：

  [J1]─k1─[J2]─k2─[J3]
  电机   联轴器  螺旋桨

状态空间方程：
  θ̈1 = (T_em - k1(θ1-θ2)) / J1
  θ̈2 = (k1(θ1-θ2) - k2(θ2-θ3)) / J2
  θ̈3 = (k2(θ2-θ3) - T_prop) / J3

固有频率（二自由度近似）：
  ω_n1² = k1(J1+J2)/(J1×J2)
  ω_n2² = k2(J2+J3)/(J2×J3)

典型参数（内河推船 500kW）：
  J_motor  = 8.5 kg·m²（厂家提供）
  J_prop   = 45 kg·m²（CFD估算）
  k_coupling = 2.5×10⁵ N·m/rad（橡胶联轴器）

  ω_n1 = √(2.5×10⁵ × (8.5+45)/(8.5×45)) = 89 rad/s → f_n1 = 14.2 Hz

  禁速区：n_critical = f_n1×60/Z = 14.2×60/4 = 213 rpm（4叶桨）
  禁速区范围：191~234 rpm（±10%）

    7.2.7 模块六：降额与故障管理模块¶

    降额逻辑状态机（Simulink Stateflow 实现）：

状态0：正常运行
  条件：T_winding ≤ 130°C AND Q_coolant ≥ Q_min AND 无故障
  输出：P_allow = P_rated（100%）

状态1：热预警
  进入条件：T_winding > 130°C 持续5s
  退出条件：T_winding < 125°C 持续60s → 状态0
  输出：P_allow = P_rated，泵升至100%转速，告警

状态2：热降额
  进入条件：T_winding > 155°C 持续10s
  退出条件：T_winding < 145°C 持续120s → 状态1
  输出：P_allow = P_rated × (170-T_winding)/(170-155)
        （线性降额，170°C时降至0）

状态3：冷却降级
  进入条件：冷却流量 < 60% 额定（主泵故障，备泵运行）
  输出：P_allow = 0.80 × P_rated

状态4：紧急停机
  进入条件：T_winding > 175°C OR 过流 > 1.5×额定
  输出：P_allow = 0，驱动使能关断，保留冷却泵

前馈优化（新增）：
  P_pred_loss = P_cmd × (1 - η_motor(n, T))
  Q_pump_pred = f(P_pred_loss)  ← 根据预测损耗提前调节泵速
  效果：热响应滞后从60s→5s，降额频率降低约80%

    7.3.2 扭振详细仿真（Simpack/ANSYS）¶

    扭振分析完整流程：

    Step 1：轴系几何建模
  - 电机转子：转动惯量J（厂家提供或用CAD计算）
  - 各轴段：GIp/L（G=80GPa，Ip=πd⁴/32，L=轴段长度）
  - 联轴器：等效扭转刚度k_c（厂家提供，含温度修正）
  - 螺旋桨：J_prop（CFD估算，加附加水质量效应）

Step 2：附加水质量效应（海船螺旋桨必须考虑）
  J_prop_wet = J_prop_dry × (1 + k_added_mass)
  k_added_mass ≈ 0.20~0.25（4叶桨经验值）
  → 水中固有频率比空气中低约10%！
  ⚠️ 反例：某项目用干态固有频率设计禁速区，
     实际螺旋桨入水后固有频率降低11%，
     禁速区设置偏高，实际共振点在"安全区"内，
     海试发现强烈振动后紧急修改控制参数。

Step 3：激励谐波分析
  电机电磁激励（PMSM）：
    主要激励阶次：6k（k=1,2,3...）→ 6倍频为最主要
    幅值：T_6th ≈ 5%×T_rated（典型值，需电磁仿真确认）

  螺旋桨水动力激励：
    激励阶次：Z（叶片数）和 Z的整数倍
    幅值：与螺旋桨不均匀来流有关（伴流场）
    海船：伴流不均匀度高 → 激励幅值大
    内河：水浅、流速均匀 → 激励幅值相对小

Step 4：Campbell图绘制
  横轴：转速（rpm），纵轴：频率（Hz）
  固有频率线：水平线（f_n1, f_n2...）
  激励线：斜线（f = order × n/60）
  交叉点 ± 10% 转速范围 = 禁速区

Step 5：强迫振动响应计算
  τ_dynamic = T_excitation × Q_factor（共振放大因子）
  Q ≈ 1/(2ζ)（ζ=阻尼比，典型0.02~0.05）
  τ_total = τ_static + τ_dynamic ≤ τ_allowable / 1.5（安全系数）

    海船扭振案例（4000kW渡船）：

    初始设计：
  联轴器刚度：5×10⁵ N·m/rad（过高）
  一阶固有频率：f_n1 = 18.5 Hz
  4叶桨交叉点：n = 18.5×60/4 = 277.5 rpm
  额定转速：280 rpm → 共振点就在额定转速附近！

解决方案：
  Option A：更换软联轴器（刚度降至2×10⁵ N·m/rad）
    f_n1降至11.8Hz，共振点=177rpm（低于最低运行转速）✅
  Option B：增加飞轮（增大J_motor）
    增重200kg，成本高，不推荐
  Option C：设置禁速区（250~305rpm）
    需要快速穿越该区域，操纵不便，不推荐

选择Option A，更换联轴器后验证：
  Campbell图无禁区交叉（额定转速280rpm以上）✅
  强迫振动响应计算：τ_dynamic/τ_static = 0.08（8%，可接受）✅

    7.3.3 电磁场仿真（Maxwell 2D/3D）¶

    主要仿真任务：

    任务1：损耗精确计算（用于热网络参数标定）

        → 验证：系统在3s内检测到双路失效，
                自动降额至60%功率（失效安全）
                故障代码正确，HMI显示正确告警
t=60s：恢复传感器A信号正常
        → 验证：系统识别A路恢复，但保持当前状态，
                等待人工确认后才能提升功率（防误操作）

验收准则：
  ① 双路失效检测时间 ≤ 3s ✅
  ② 降额至60%的过渡时间 ≤ 5s（无冲击）✅
  ③ 故障代码：0x2031（温度传感器双路失效）✅
  ④ HMI显示："温度监测失效，推力降级运行，请联系维修" ✅
  ⑤ 功率稳定在60%额定，无振荡 ✅

    7.6 仿真与试验相关性标定¶

    7.6.1 标定流程（三步法）¶

    Step 1：台架试验采集基准数据
  试验工况矩阵（至少覆盖以下组合）：

  负载点：30% / 50% / 75% / 100% 额定功率
  冷却流量：最小/额定/最大流量
  环境温度：15°C / 25°C / 35°C（模拟不同海温）

  测量量：
  - 绕组温升（每相，PT100）
  - 冷却套进出口温差
  - 损耗（输入功率-输出功率，精度0.1%）
  - 各点压差和流量
  - 振动加速度（轴承座）

Step 2：模型参数辨识
  热网络参数辨识：
    R_winding_slot：由绕组-铁芯温差 ÷ 铜耗估算
    R_yoke_jacket ：由铁芯-冷却套温差 ÷ 铁耗估算
    C_thermal    ：由温升时间常数 τ = R×C 拟合

  优化方法：最小二乘法拟合仿真曲线到试验数据
  目标函数：min Σ(T_sim_i - T_test_i)²

Step 3：标定验证（独立验证工况）
  选取与标定工况不同的3个验证工况
  验收准则：
    温升预测误差 ≤ ±5°C（L3模型）
    损耗预测误差 ≤ ±2%（L2模型）
    固有频率预测误差 ≤ ±5%（扭振模型）

    7.9 典型工程案例¶

    7.9.1 内河客船案例：仿真发现换热器选型错误¶

    背景：某内河电推客船，2×1000kW双轴推进，设计阶段仅进行热计算，
      未做动态仿真，换热器按单一额定工况选型。

问题发现（仿真阶段）：
  仿真工况：靠泊→出港→巡航→最大速度完整任务剖面动态仿真

  发现1：靠泊低功率（5%额定）阶段，冷却液温度降至28°C，
         换热器出力不足（偏低），泵工作点严重偏离高效区
         泵效率：42%（设计值78%）→ 泵长期欠载运行，寿命降低

  发现2：从靠泊→最大功率（5min内功率从5%到100%），
         热响应滞后约8分钟，绕组温升峰值92K（超过F级80K！）
         静态仿真显示最终稳态78K（看似安全），
         但动态仿真暴露了瞬态峰值超标问题。

解决方案：
  ① 换热器缩小15%面积（按等效任务剖面功率选型）
  ② 泵变频调速（根据功率指令调节泵速）
  ③ 前馈控制：功率指令发出同时预调泵速，热响应从8min→2min

成本分析：
  仿真发现时：增加变频泵控制器约3万元，换热器优化节省1.5万元
  台架试验发现：需重新选型换热器+重新安装，约15万元+延期1个月
  海试发现：返厂修改，约80万元+延期3个月
  → 仿真阶段发现节省约78万元工程损失

    7.9.2 海船案例：扭振仿真避免轴系共振¶

    背景：某4000kW海船电推，设计转速280rpm，4叶桨。
      项目方认为"参考同类船型不做扭振分析"。

仿真分析（要求补做）：
  一阶固有频率（考虑附加水质量）：f_n1 = 19.2 Hz
  4叶桨激励频率：f_exc = 4×280/60 = 18.7 Hz ≈ f_n1！
  → 额定转速附近存在严重共振风险！

如果不做仿真直接投产：
  预计结果：强烈轴系振动，联轴器橡胶元件1~2个月撕裂，
            振动超过船级社限值，无法通过海试

解决方案（仿真后提前处理）：
  联轴器刚度降低35%→ f_n1=15.1Hz，共振点降至227rpm
  电控增加225~232rpm禁速区（穿越时间<3s）

节省成本：
  设计变更（联轴器重选）：约2万元，提前2周处理
  vs 投产后返工：整套轴系拆解+重装+海试，约200万元+延期2个月
