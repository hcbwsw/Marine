# 能耗优化策略

## 1. 航行优化

### 1.1 经济航速

**核心原理**：排水型船舶 P ∝ V³，降低航速可大幅减少能耗。

| 航速比 | 功率比 | 单位里程能耗比 | 续航增幅 |
|--------|--------|---------------|----------|
| 100% | 100% | 100% | 基准 |
| 90% | 73% | 81% | +23% |
| 80% | 51% | 64% | +56% |
| 70% | 34% | 49% | +104% |

📌 **实操建议**：在非紧急情况下，降低 10-20% 航速是最简单有效的节能措施。

### 1.2 航线优化

- **顺流航行**：利用水流速度，减少相对水速度
- **避流行驶**：避开强逆流区域
- **浅水规避**：水深 < 1.5T 时阻力增加 10-40%
- **风浪规避**：顶风顶浪增加 15-30% 功率

### 1.3 纵倾优化

- 船舶纵倾影响阻力
- 通过调整压载水或货物分布，找到最优纵倾角
- 一般微首倾（0.1-0.3m）阻力最小

## 2. 系统优化

### 2.1 电池 SOC 工作区间

| SOC 区间 | 效率 | 寿命影响 | 建议 |
|----------|------|----------|------|
| 90-100% | 充电效率低 | 加速老化 | 避免长期满充 |
| 50-90% | 最优区间 | 寿命最长 | **推荐工作区间** |
| 20-50% | 正常 | 正常 | 正常使用 |
| 10-20% | 正常 | 轻微影响 | 尽量避免 |
| < 10% | 放电效率低 | 严重损害 | 禁止 |

💡 **EMS 策略**：日常运营保持 SOC 在 30-80% 区间，既保证续航又延长电池寿命。

### 2.2 电机工作点优化

电机效率随负荷变化：

```
效率
  │        ╭─────╮
  │      ╱         ╲
  │     ╱           ╲
  │    ╱             ╲
  │   ╱               ╲
  │  ╱                 ╲
  │ ╱                   ╲
  └──────────────────────── 负荷率
  0%   25%   50%   75%  100%
```

- 电机在 50-90% 负荷率时效率最高
- < 30% 负荷率时效率急剧下降
- **优化措施**：多电机配置时，低负荷可停一台电机

### 2.3 辅机用电管理

| 设备 | 功率 | 优化措施 |
|------|------|----------|
| 空调 | 3-15 kW | 变频控制，温度设定 26°C |
| 照明 | 1-3 kW | 全部 LED，分区控制 |
| 舱底泵 | 1-5 kW | 液位自动控制，避免空转 |
| 消防泵 | 5-15 kW | 定期试运转，平时关闭 |
| 导航通信 | 0.5-1 kW | 正常开启 |

📌 辅机用电占总能耗 10-25%，优化后可节省 5-10% 总能量。

## 3. 维护优化

### 3.1 船体清洁

- 生物附着（藤壶、藻类）增加阻力 10-30%
- 建议每 6-12 个月进坞清洁
- 使用防污漆可延长清洁周期
- 水下机器人清洗（新兴技术）

### 3.2 螺旋桨维护

- 桨叶抛光：粗糙度降低可减少阻力 3-5%
- 桨叶修复：缺损、变形需及时修复
- 定期检查：空化腐蚀、裂纹

### 3.3 电机保养

- 轴承润滑：按厂家要求定期更换润滑脂
- 绝缘检测：定期测量绝缘电阻
- 冷却系统：清洁冷却器，保证散热
- 振动监测：异常振动及时排查

### 3.4 电池健康监测

- 定期检查单体电压一致性
- 监测内阻变化趋势
- 温度分布均匀性检查
- 连接器紧固状态检查
- SOH 定期评估，规划更换时间
