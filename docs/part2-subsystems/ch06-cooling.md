## 4.1 热设计链路（完整五步流程）

### Step 1：损耗计算（热源识别）

```
铜耗（主要热源，随负载变化）：
  P_cu = 3 × I² × R_phase
  示例：额定电流500A，相电阻R=0.01Ω
        P_cu = 3 × 500² × 0.01 = 7500W = 7.5kW

铁耗（近似恒定，随频率变化）：
  P_fe = K_h × f × B_m^α × V_core   （磁滞损耗）
       + K_e × f² × B_m² × V_core   （涡流损耗）
  典型值：约占额定铜耗的20~40%

机械损耗（风摩耗，随转速变化）：
  P_mech = k_mech × n^1.5
  典型值：约占额定功率的0.5~1.5%

总热源（额定工况）：
  Q_total = P_cu + P_fe + P_mech
  典型范围：额定功率的4~8%

  示例：2000kW电机，效率96%
        Q_total = 2000 × (1-0.96) = 80kW
        其中：铜耗约55kW，铁耗约20kW，机械损耗约5kW

⚠️ 反例：某项目损耗计算仅按"额定功率×(1-效率)"估算，
   忽略了铁耗在低负载下占比更大的特性。
   结果：轻载工况（30%负载）下电机温升超预期，
   因为铁耗并未等比例下降，导致相对散热需求更高。
   正确做法：分别计算铜耗和铁耗，在任务剖面每个工况段验证。
```

### Step 2：换热系统设计（ε-NTU 法）

```
ε-NTU 法基本公式：
  传热效能：ε = Q_actual / Q_max
           Q_max = C_min × (T_hot_in - T_cold_in)
  传热单元数：NTU = UA / C_min
  对于逆流换热器（推荐）：
    ε = [1 - exp(-NTU×(1-Cr))] / [1 - Cr×exp(-NTU×(1-Cr))]
    其中 Cr = C_min/C_max（热容量比）

设计目标（典型内河客船，T_sea_max=35°C）：
  冷却液（热侧）：入口65°C → 出口45°C，ΔT=20°C
  海水（冷侧）：入口35°C → 出口50°C，ΔT=15°C
  目标效能：ε ≥ 0.85

流量计算（基于能量守恒）：
  Q = ṁ_coolant × Cp_coolant × ΔT_coolant
  ṁ = Q / (Cp × ΔT) = 80000 / (3800 × 20) ≈ 1.05 kg/s ≈ 63 L/min
  → 加30%裕度（考虑污堵+管路分配不均）：
    设计流量 ≈ 82 L/min（内+外循环合计）

换热面积计算：
  Q = U × A × ΔT_lm
  ΔT_lm（对数平均温差，逆流）= (ΔT₁-ΔT₂) / ln(ΔT₁/ΔT₂)
       = (65-50 - 45-35) / ln(15/10) = 12.3°C
  A = Q / (U × ΔT_lm) = 80000 / (3000 × 12.3) ≈ 2.17 m²
  加污堵裕度（RF=0.0001 m²·K/W → U修正≈85%）：
    A_design = 2.17 / 0.85 ≈ 2.55 m²

板式换热器板数估算：
  单板面积典型值：0.1~0.5 m²
  若单板0.3 m²：板数 ≈ 2.55/0.3 ≈ 9板 → 选10板（取整留裕度）
```

### Step 3：内循环管网水力计算

```
系统压降组成：
  ΔP_total = ΔP_hx + ΔP_jacket + ΔP_pipe + ΔP_valve + ΔP_fitting

示例计算（额定流量63 L/min下）：
  ΔP_hx    = 20 kPa   （换热器压降，厂家提供Q-ΔP曲线）
  ΔP_jacket = 25 kPa  （电机冷却套，厂家提供）
  ΔP_pipe   = 15 kPa  （管路沿程+局部阻力）
  ΔP_valve  = 10 kPa  （调节阀 + 过滤器）
  ΔP_total  = 70 kPa

泵扬程选择（加10%裕度）：
  H_pump ≥ ΔP_total / (ρg) × 1.1
         = 70000 / (1000 × 9.81) × 1.1
         ≈ 7.85 m → 选 H=10m（符合标准泵型号）

泵工作点验证：
  Q_pump(H=10m) vs Q_design(H=7.85m)
  工作点须在泵效率曲线 η_pump ≥ 70% 区间内
  📐 绘图工具：Excel/MATLAB绘制泵曲线+系统曲线，交点即工作点

系统阻力曲线拟合：
  ΔP_system = R_hydraulic × Q²
  R_hydraulic = ΔP_total / Q_design²
              = 70000 / (0.00105)² ≈ 6.35×10¹⁰ Pa/(m³/s)²
```

### Step 4：NPSH 校核（防汽蚀）

```
NPSH_a（有效汽蚀余量）计算：
  NPSH_a = (P_abs - P_vapor) / (ρg) + V²/(2g) - H_loss_suction

  其中：
  P_abs   = P_atm + P_static（泵入口绝对压力）
  P_vapor = 乙二醇溶液饱和蒸汽压（50°C时约12 kPa）
  H_loss_suction = 泵入口管路阻力损失

内循环泵NPSH校核示例：
  P_atm = 101.3 kPa，膨胀箱正压 P_static = 10 kPa
  P_vapor = 12 kPa（50°C乙二醇溶液）
  H_loss_suction ≈ 1.5 m
  NPSH_a = (101300 + 10000 - 12000) / (1000 × 9.81) - 1.5
         ≈ 10.1 - 1.5 = 8.6 m
  要求：NPSH_a ≥ 1.3 × NPSH_r
        若泵NPSH_r = 3.0m → 1.3 × 3.0 = 3.9m < 8.6m ✅

外循环泵（海水侧）特殊注意：
  满载吃水时静压头可能低于轻载，需两种工况均校核
  热带海水密度：1025 kg/m³（略高于淡水，影响ρg）
  要求：NPSH_a ≥ 1.5 × NPSH_r（更严格，考虑海况变化）

⚠️ 反例（来自手册v3.0原版）：
   某项目忽略满载吃水时静压头降低0.8m，
   导致NPSH_a不足，海试出现汽蚀噪声，被迫降速运行。
   整改：重新选泵（选NPSH_r更小的型号）+ 安装降噪喷嘴。
   代价：5万元整改费 + 2周停航。
   预防：设计时两种吃水工况均要校核！
```

### Step 5：膨胀水箱容积计算

```
膨胀水箱容积计算：
  V_tank ≥ 15% × V_system_total

  系统总液量估算：
    电机冷却套：≈3L
    内循环管路（内径25mm，总长约15m）：≈7L
    换热器液侧：≈2L
    膨胀水箱安全余量：≈3L
    V_system_total ≈ 15L（典型内河船200kW电机）

  → V_tank ≥ 15% × 15 = 2.25L → 取4L（含气腔，防过充满）

  乙二醇溶液热膨胀量验证：
  ΔV = α_乙二醇 × V_system × ΔT
      = 0.00065 × 15 × (65-20)  ← 温度范围：停机时20°C → 运行时65°C
      = 0.44L → 4L容积足够容纳热膨胀 ✅

三级液位联锁设定：
  液位状态     容积百分比    控制动作
  ───────────────────────────────────────
  正常          >80%        无动作
  低液位        60~80%      报警，提示补液
  极低液位      40~60%      降额至80%额定功率
  极极低液位    <40%        保护停机（防泵空转汽蚀）
  
  响应时间要求：极极低液位触发停机 ≤ 2s
```

## 4.3 冷却系统拓扑决策矩阵

| 方案 | 优势 | 劣势 | 适用场景 | 成本比 |
|-----|------|------|---------|-------|
| 单回路海水直冷 | 结构简单、体积小 | 电机需耐腐蚀、维护频繁、污堵快 | ≤100kW 内河船（成本优先）| 1× |
| 双回路（内乙二醇+外海水）| 电机保护好、可靠性高、水质可控 | 增加换热器和管路复杂度 | **推荐方案**（中大功率）| ~1.3× |
| 双回路+备泵冗余 | 冗余度高、A≥99.9% | 成本最高、重量大 | 海船/客船等高可用要求 | ~1.6× |
| 风冷+液冷混合 | 适合低温环境，防冻简单 | 控制复杂、散热能力受环境限制 | 北方内河冬季极寒运行 | ~1.4× |

## 4.4 冗余设计策略与切换逻辑

### 内循环泵冗余切换完整逻辑

```
硬件配置：
  主泵（变频调速）：正常运行
  备泵（变频调速）：热备（通电待机，0转速）
  管路配置：主泵+备泵并联，各带单向止回阀（防倒流）

自动切换触发条件（满足任一即切换，逻辑OR）：
  条件1：主泵出口压力 < 设定值下限（持续5s）
         设定值 = 正常工作点压力 × 70%
  条件2：主泵驱动器故障信号
  条件3：主泵电机电流 > 1.2×额定电流（持续3s，可能叶轮卡塞）
  条件4：流量传感器读数 < 60%额定流量（持续5s）

切换时序（目标：切换时间≤5s）：
  t=0.0s：故障触发判定（任一条件满足）
  t=0.5s：HMI发出黄色告警"主泵故障，切换备泵"
  t=1.0s：备泵启动指令发出（变频器斜坡升速）
  t=3.0s：备泵达到额定转速（变频升速完成）
  t=4.0s：确认备泵出口压力正常
  t=5.0s：主泵隔离阀关闭（电动阀，防主泵反转）
  t=5.0s：控制器记录"备泵运行"事件，上报船舶管理系统

降级运行能力（单泵运行）：
  单泵最大流量 = 额定双泵流量的70%（泵并联特性）
  → 控制器自动降额至80%额定推进功率（防过热）
  → HMI 显示"冷却降级运行，推进功率限制80%"

⚠️ 典型事故案例（来自手册v3.0）：
   某电推船未安装备泵，主泵轴封泄漏→冷却液渗入舱底。
   电机温度上升→保护停机→在长江主航道失去动力漂流。
   事后分析：增加约2万元备泵，可完全避免此次事故。
   结论：内河客船必须配备备泵，货船强烈建议配备。
```

### 外循环泵冗余策略

```
海船/客船：
  主备泵各一台 + 并联旁通阀（防滤器严重堵塞时旁路运行）
  旁通运行条件：外循环压差 > 1.5× 初始值（表明滤器严重堵塞）
  旁通后：直接进海水，换热器仍有效，但无过滤→加强清洗频率

内河船（成本优先）：
  可接受单泵 + 定期清洗维护策略
  但必须满足：
    ① 换热器具备CIP在线清洗接口（可不停机清洗）
    ② 滤器为双联可切换式（可在线切换清洗）
    ③ 故障备件（叶轮+轴封）随船携带
```

## 4.5 海水外循环特殊设计要求

### 防腐蚀材料选择

```
海水侧接触材料（按优先级）：
  首选：钛合金 Gr.2（最佳，但贵）
  次选：双相不锈钢 SAF 2205（性价比高）
  可用：铜镍合金 90/10（CuNi10Fe）（传统船用材料）
  禁止：304不锈钢（点蚀风险极高，使用寿命<2年）
        316L不锈钢（仍有点蚀风险，不推荐海水直接接触）

电偶腐蚀防护：
  不同金属直接接触→电偶腐蚀（活性金属优先腐蚀）
  防护措施：绝缘法兰垫片 + 绝缘套管（螺栓位置）
  检测：安装牺牲阳极（锌块），定期更换（通常每年一次）

结构腐蚀监测（推荐）：
  外循环管路关键段安装腐蚀挂片
  每半年取样分析（腐蚀速率>0.1mm/年→必须整改）
```

### 海生物防污

```
进水滤器：
  粗滤（进水口）：网目 ≤ 5mm（防贝类/鱼虾进入）
  细滤（泵前）：网目 ≤ 2mm（防微小生物堵塞换热器）
  推荐：双联可切换式，支持在线切换清洗（不停机）

换热器防污：
  预留反冲洗接口：正常运行时水流方向A→B；反冲洗时切换B→A
  清洗频率（根据航区）：
    内河（泥沙多）：每月一次反冲洗
    近海（生物多）：每季度一次反冲洗 + 每年一次化学清洗
    热带海域（生物繁殖旺盛）：每月一次反冲洗

进水口位置要求：
  距船底 ≥ 300mm（防沉积物/泥沙吸入）
  远离排污口 ≥ 3m（防污水/热水回流）
  远离螺旋桨 ≥ 1D（防气泡吸入导致汽蚀）
  开口方向：朝后或侧向（航行时有正压进水，提高流量）
```

### 冰区航行特殊要求（北方船舶）

```
防冻设计：
  进水管：电伴热（功率约50W/m），停机后自动开启
  换热器：停机程序包含"海水侧自动排空"步骤
          排空气源：专用气压管路或高压氮气瓶
  最低设计温度：按航区设定，北方内河最低按-40°C设计

防冻液管理：
  乙二醇浓度监测：每季度抽样检测冰点（目标≤-35°C）
  乙二醇寿命：约3~5年更换一次（防腐蚀添加剂失效）
  密度计随船配备（现场快速检测乙二醇浓度）
```

## 4.6 换热器选型详细指南

### 换热器类型选择

| 换热器类型 | 适用场景 | 优点 | 缺点 | 适用功率范围 |
|----------|---------|------|------|------------|
| 板式换热器（PHE）| 空间受限、效率要求高 | 换热效率高（ε≥0.90）、体积小、可拆洗 | 承压能力有限（通常≤2.5MPa）、密封件需维护 | ≤5000kW |
| 壳管式换热器（STH）| 高压、海洋污染重 | 承压能力强、结构简单 | 体积大、换热效率较低（ε≈0.70~0.85）| 全范围 |
| 螺旋板式换热器 | 内河船（含固体颗粒）| 不易堵塞 | 成本高、维护复杂 | ≤2000kW |
| 管套管换热器 | 小功率（≤200kW）| 结构最简单 | 效率低、体积大 | ≤200kW |

**推荐组合：中大功率船舶使用板式换热器（主推），重污染航区备选壳管式。**

### 换热器选型核查清单

```
□ ε-NTU计算完成，换热效能ε≥0.85（含污堵裕度后）
□ 污堵裕度系数：RF≥0.0001 m²·K/W（板换）/ RF≥0.00015（壳管）
□ 海水侧材质：钛合金或双相不锈钢，不接受304/316L
□ 工作压力：设计压力≥1.5×最大工作压力（安全系数）
□ 进出口法兰：与管路规格匹配，DN/PN等级确认
□ 反冲洗接口：已预留（内河/近海必须）
□ CIP接口：已预留（支持不停机在线清洗，海船推荐）
□ 安装方向：竖向安装（板换），便于自然排液，避免积液腐蚀
□ 厂家FAT：压力试验（1.5×设计压力，30min无泄漏）
□ 性能FAT：换热性能试验（在规定流量/温度下，验证ε值）
```

## 4.7 冷却系统传感器配置

```
传感器配置原则：
  关键参数（影响保护动作）：三传感器投票（2/3判决）
  重要参数（影响告警）：双传感器（任一触发告警）
  一般参数（监测用）：单传感器（故障时进入估算模式）

推荐传感器清单：
┌──────────────────────────────────────────────────────────┐
│ 测量位置         │ 传感器类型   │ 数量 │ 用途           │
├──────────────────────────────────────────────────────────┤
│ 电机绕组（每相）  │ PT100       │ 3×3  │ 热保护（三投票）│
│ 冷却液入口温度   │ PT100       │ 2    │ 冷却性能监测   │
│ 冷却液出口温度   │ PT100       │ 2    │ 冷却性能监测   │
│ 内循环压力      │ 4-20mA压力表 │ 2    │ 泵工作点监测   │
│ 内循环流量      │ 脉冲/电磁   │ 1    │ 冷却能力判断   │
│ 膨胀箱液位      │ 开关量（三级）│ 1套  │ 液位保护联锁   │
│ 外循环压差      │ 差压变送器   │ 1    │ 滤器污堵监测   │
│ 外循环流量      │ 电磁流量计   │ 1    │ 散热能力监测   │
│ 推力轴承温度    │ PT100       │ 2    │ 轴承状态监测   │
└──────────────────────────────────────────────────────────┘

三传感器投票逻辑（绕组温度）：
  3个PT100传感器读数：T1, T2, T3
  中间值选择：T_winding = median(T1, T2, T3)
  偏差监测：若任一 |Ti - T_winding| > 5°C → 传感器异常告警
  单传感器故障：剩余两个取均值（需HMI告警提示"降级监测模式"）
  两传感器故障：进入"传感器失效保护模式"（降额至50%，等待检修）
```

## 4.8 冷却系统调试与标定流程

```
台架调试阶段（D4阶段，对应第20章台架试验）：

Step 1：冷态充液与排气
  充注50%乙二醇溶液至膨胀箱正常液位
  低速循环泵运行，从最高点排气阀排出气泡
  观察：液位稳定、无气泡出口 → 排气完成（通常需30~60min）

Step 2：泵特性标定
  测量不同转速下的Q-H曲线（10个工作点）
  与厂家曲线对比：误差 ≤ 5%（流量）/ ≤ 3%（扬程）
  标定结果用于热网络模型参数修正

Step 3：换热器性能标定
  在规定流量和温度条件下，测量换热效能ε_actual
  ε_actual vs ε_design 误差 ≤ 5%
  若误差过大（>10%）：检查安装朝向/排气是否完整/流量是否正确

Step 4：热网络模型标定（对应第10章）
  施加阶跃热负荷（电机通电），记录各点温升曲线
  调整热网络模型参数（R_th, C_th）
  目标：仿真与实测温升误差 ≤ 5°C（所有测量点）

Step 5：三级热保护阈值验证
  逐步升温至每级阈值，验证：
    Level 1（130°C）：泵转速100%，记录告警 → 通过/不通过
    Level 2（155°C）：降额动作，P_output线性下降 → 通过/不通过
    Level 3（175°C）：停机 ≤ 0.5s响应时间 → 通过/不通过
  所有阈值验证通过后，在VCRM中关闭对应验证条目
```

## 4.10 本章小结与关键输出物

**本章核心要点（五条）：**

1. **热设计五步流程不可省略**
   损耗计算→换热计算→水力计算→NPSH校核→膨胀水箱，每步都有量化准则，跳步会导致设计漏洞。

2. **换热器按加权等效工况选型，非峰值功率**
   任务剖面中经济航速占60%时长，按此工况选型换热器；峰值仅验证短时能力。

3. **NPSH校核必须覆盖两种吃水工况**
   满载吃水=静压头降低=NPSH_a减小。遗漏满载校核是最常见的冷却系统设计缺陷之一。

4. **备泵是最高性价比的可靠性措施**
   2万元备泵 vs 百万元停航损失，ROI极高。内河客船必须配备，货船强烈建议。

5. **传感器投票机制防止误触发和漏保护**
   单传感器故障导致误停机（误触发）或假正常（漏保护）都会造成严重后果。三传感器中间值选择是工程最优解。

**关键输出物：**
```
□ 热计算报告（五步流程，含各工况计算表格）

□ 管网水力计算书（系统压降+泵工作点验证图）
□ NPSH校核计算书（轻载+满载两种吃水工况）
```
