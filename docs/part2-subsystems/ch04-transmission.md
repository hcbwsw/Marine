# 传动系统设计

## 1. 传动方式对比

### 1.1 直驱（Direct Drive）

```
电机 → 联轴器 → 推力轴承 → 螺旋桨轴 → 螺旋桨
```

| 特点 | 说明 |
|------|------|
| 效率 | 最高（无齿轮损耗，98-99%） |
| 电机要求 | 低转速大扭矩（100-500 rpm） |
| 体积重量 | 电机大且重 |
| 维护 | 最简单 |
| 噪音 | 最低 |
| 适用 | 大直径低转速桨，空间充裕 |

### 1.2 齿轮箱减速传动

```
电机 → 联轴器 → 齿轮箱 → 推力轴承 → 螺旋桨轴 → 螺旋桨
```

| 特点 | 说明 |
|------|------|
| 效率 | 95-98%（齿轮损耗 2-5%） |
| 电机要求 | 高转速（1000-3000 rpm） |
| 体积重量 | 电机小而轻，但增加齿轮箱 |
| 维护 | 需定期换油、检查齿轮 |
| 噪音 | 齿轮啮合噪音 |
| 适用 | 空间受限，高速桨 |

### 1.3 皮带传动

```
电机 → 皮带轮 → 皮带 → 皮带轮 → 螺旋桨轴
```

- 结构简单，成本低
- 效率 95-97%
- 适合小功率（< 50 kW）
- 有打滑风险，需定期检查张紧度

## 2. 轴系设计要点

### 2.1 轴径计算

**按扭矩计算最小轴径：**

$$d \geq \sqrt[3]{\frac{16T}{\pi \tau_{allow}}}$$

其中：
- d：轴径（mm）
- T：传递扭矩（N·mm）
- τ_allow：许用剪切应力（MPa）

**按 CCS 规范：**

$$d \geq k \times \sqrt[3]{\frac{P}{n}}$$

其中：
- k：材料系数（碳钢 k=100-110，不锈钢 k=90-100）
- P：传递功率（kW）
- n：转速（rpm）

📌 **注意**：电推进船舶轴系需考虑电机峰值扭矩（可达额定 2 倍），轴径需按峰值扭矩校核。

### 2.2 轴承间距

- 轴承间距过大 → 轴挠度大，振动加剧
- 轴承间距过小 → 轴承负荷集中
- 一般推荐：L/d = 10-15（L 为轴承间距，d 为轴径）

### 2.3 对中要求

| 项目 | 允许偏差 |
|------|----------|
| 径向位移 | ≤ 0.05 mm |
| 角度偏差 | ≤ 0.05 mm/m |
| 轴向间隙 | 按联轴器规格 |

⚠️ 电推进船舶对中要求更高——电机对振动敏感，对中不良会导致轴承过早失效。

### 2.4 联轴器选型

| 类型 | 特点 | 适用 |
|------|------|------|
| 刚性联轴器 | 结构简单，无补偿能力 | 短轴系，对中精度高 |
| 弹性联轴器 | 可补偿少量偏差，减振 | 推荐用于电推进 |
| 万向联轴器 | 可补偿大角度偏差 | 舵桨、特殊布置 |
| 膜片联轴器 | 无背隙，高扭矩 | 高精度传动 |

💡 **推荐**：电推进系统优先选用弹性联轴器，可吸收电机与轴系之间的振动和少量不对中。

### 2.5 尾轴密封

| 类型 | 特点 | 适用 |
|------|------|------|
| 填料密封 | 结构简单，需定期压紧 | 小型船舶 |
| 机械密封 | 密封性好，寿命长 | 推荐 |
| 唇形密封 | 结构紧凑，成本低 | 小型船舶 |
| 水润滑密封 | 无油污染，环保 | 环保要求高 |

## 3. 效率与损耗

### 3.1 各环节损耗

| 环节 | 效率 | 损耗来源 |
|------|------|----------|
| 联轴器 | 0.99 | 弹性体变形 |
| 齿轮箱 | 0.95-0.98 | 齿面摩擦、搅油 |
| 轴承 | 0.98-0.99 | 滚动摩擦 |
| 密封 | 0.99 | 摩擦损耗 |
| **轴系总效率** | **0.93-0.97** | |

### 3.2 振动分析

**轴系振动来源：**
- 电机电磁振动（齿槽效应、PWM 谐波）
- 螺旋桨水动力激励（叶频振动）
- 轴系不平衡/不对中
- 齿轮啮合振动

**减振措施：**
- 弹性联轴器
- 减振基座
- 轴系动平衡
- 螺旋桨优化（叶数、侧斜）
