## 5.1 螺旋桨–电机匹配分析（设计起点）

**这是传动系统设计的第一步，必须首先完成，其结果决定电机选型和控制带宽设计上限。**

### 螺旋桨负载特性（定距桨 FPP）

```
定距桨（FPP）负载特性方程：
  转矩：T_prop = K_Q × ρ × n² × D⁵
  推力：F_prop = K_T × ρ × n² × D⁴
  功率：P_prop = 2π × K_Q × ρ × n³ × D⁵
  注意：功率∝n³（转速三次方）→ 转速降10%，功率降约27%！

  其中：
  K_T、K_Q = 螺旋桨敞水特性系数（需CFD计算或系列桨查表）
  ρ         = 水密度（淡水1000 kg/m³ / 海水1025 kg/m³）
  D         = 桨径（m）
  n         = 转速（r/s，注意：非rpm！）

示例计算（D=1.2m，n=5 r/s=300 rpm，K_Q=0.03，海水）：
  T_prop = 0.03 × 1025 × 5² × 1.2⁵
         = 0.03 × 1025 × 25 × 2.488 ≈ 1912 N·m
  P_prop = 2π × 1912 × 5 ≈ 60 kW
```

### 匹配设计四个关键要求

```
要求1：额定点转矩匹配（含裕度）
  T_motor_rated ≥ T_prop(n_design) × 1.15
  示例：T_prop=1.91 kN·m → T_motor_rated ≥ 1.91 × 1.15 = 2.20 kN·m

要求2：全转速范围包络（T-n曲线包住所有工况点）
  验证方法：在T-n图上同时画出：
    ① 电机T-n特性曲线（额定区+过载区）
    ② 螺旋桨在各工况（正车/倒车/靠泊）的负载点
    ③ 确认所有负载点在电机T-n曲线包络内

要求3：倒车校核（四象限运行能力）
  倒车转矩 ≈ 正车额定转矩 × 80~85%（定距桨）
  变频器必须支持四象限运行（能量可以回馈）
  倒车加速斜率限制：避免机械冲击 → 限制在5~10rpm/s

要求4：堵转工况保护（螺旋桨缠绕异物）
  瞬时堵转转矩可达额定3~5倍
  保护措施：变频器过流保护（≤1.5倍额定电流，100ms内跳闸）
            + 机械安全离合器（扭矩限制器，超载时打滑保护）
  ⚠️ 反例：某推船未设堵转保护，螺旋桨缠绕渔网后
     电机过流烧毁（绕组绝缘击穿），维修费用约18万元，
     停航15天。正确做法：软件过流保护+硬件扭矩限制器双重保护。
```

### 定距桨 vs 可调桨选型对比

| 比较维度 | 定距桨（FPP）| 可调桨（CPP）|
|---------|------------|------------|
| 推进效率（设计点）| 高 | 略低（桨毂损失约2%）|
| 低速/靠泊效率 | 低（需电机变速）| 高（桨距可调）|
| 系统复杂度 | 低 | 高（液压系统+控制）|
| 成本 | 低 | 高（约2~3倍）|
| 与变频器配合 | 需全范围变频 | 可恒速运行（简化变频器）|
| 适用场景 | 固定航线、航速变化小 | 港作拖轮、频繁变速 |

**结论：固定航线首选定距桨+变频直驱。复杂工况（港作拖轮）可考虑CPP。**

## 5.2 轴系扭振分析（船级社强制要求）

**船级社规定：扭振分析报告必须作为审图材料提交，禁止在临界转速 ±10% 范围内连续运行。**

### Step 1：轴系集总参数建模

```
将轴系离散为质量-弹簧-阻尼单元链：

元素            转动惯量 J(kg·m²)              扭转刚度 k(N·m/rad)
────────────────────────────────────────────────────────────────
电机转子         J_rotor（厂家提供）              —（刚体）
联轴器           J_coupling（查选型手册）          k_c（★关键参数！厂家提供）
推力轴承         通常并入相邻轴段计算             —
螺旋桨轴段       J = ρ_steel×π/32×d⁴×L         k = G×Ip/L
                 d=轴径(m)，L=段长(m)            G=80GPa（钢），Ip=π/32×d⁴
螺旋桨           J_prop（CFD计算或经验公式）       —（刚体）
                 经验公式：J_prop ≈ 0.3×m_prop×R_prop²

典型数值示例（内河船2000kW，300rpm）：
  J_rotor   ≈ 25 kg·m²     （电机厂家规格书）
  J_coupling ≈ 2 kg·m²     （弹性联轴器产品手册）
  k_coupling ≈ 1.5×10⁶ N·m/rad （弹性联轴器关键设计变量）
  J_shaft ≈ 8 kg·m²        （轴段计算）
  k_shaft ≈ 4×10⁷ N·m/rad （轴段计算）
  J_prop  ≈ 120 kg·m²      （CFD计算，螺旋桨附水质量修正）

⚠️ 附水质量修正（螺旋桨）：
   水中螺旋桨的等效转动惯量 J_prop_water ≈ 1.2~1.5 × J_prop_dry
   忽略附水质量 → 固有频率计算偏高约10~15% → 禁速区位置误差
   严重后果：禁速区设置偏差 → 在真实共振区内运行！
```

### Step 2：求解固有频率

```
特征方程（无阻尼自由振动）：
  [K - ω²M]{θ} = {0}

  K：扭转刚度矩阵（三对角矩阵）
  M：转动惯量对角矩阵
  ω：角频率（rad/s）

  解得各阶固有角频率 ω_n1, ω_n2...
  对应固有频率 f_ni = ω_ni / (2π)  (Hz)

典型结果（上述示例参数）：
  一阶固有频率 f_n1 ≈ 18~25 Hz
  （主要受联轴器刚度和螺旋桨惯量影响）

计算工具推荐：
  ① MATLAB/Simulink：手动建矩阵，eig()求解
  ② Simpack / AVL Excite：专业扭振分析软件（船级社认可）
  ③ Excel：仅适用于2~3质量块简化系统（工程估算用）
```

### Step 3：激励源识别与 Campbell 图

```
激励源分析（PMSM 永磁同步电机）：

激励类型1：电磁转矩脉动
  6相绕组（最常见）：激励阶次 = 6k（k=1,2,3...）
  实际激励频率 f_em = k × 6 × (n/60)
  示例：300rpm运行时，6阶激励频率 = 6 × 300/60 = 30 Hz

激励类型2：螺旋桨水动力激励
  激励阶次 = Z × k（Z=叶片数，k=1,2...）
  激励频率 f_prop = Z × (n/60)
  示例：4叶桨，300rpm → f_prop1 = 4 × 300/60 = 20 Hz

Campbell 图绘制（转速 vs 频率）：
  X轴：转速范围 0 ~ n_max
  Y轴：频率（Hz）

  水平线：轴系固有频率 f_n1, f_n2...（不随转速变化）
  斜线（从原点出发）：各阶激励频率线
    f_em_6   = 6 × n/60
    f_em_12  = 12 × n/60
    f_prop_4 = 4 × n/60（4叶桨）

  交叉点（斜线与水平线相交处）= 共振转速
  禁速区 = 每个交叉点 ± 10%转速范围

典型分析结果（示例）：
  f_n1=22Hz，4叶桨激励线交叉点 n=330rpm
  → 禁速区：297~363rpm

  f_n1=22Hz，6阶电磁激励线交叉点 n=220rpm
  → 禁速区：198~242rpm

  最终禁速区：198~242rpm 和 297~363rpm
  控制系统设定快速穿越（加速度≥5rpm/s）
```

### Step 4：强迫振动响应分析（含阻尼）

```
阻尼比典型值：
  橡胶弹性联轴器：ζ = 0.05~0.15（较高阻尼）
  硬性金属联轴器：ζ = 0.01~0.03（低阻尼，不推荐电推系统使用）
  硅油扭振减震器：ζ = 0.10~0.25（专用减震器）

动态扭应力校核（额定工况）：
  τ_dynamic = T_dynamic / W_p
  W_p = π/16 × d³（抗扭截面模量）

  强度条件：τ_static + τ_dynamic ≤ τ_allowable / 安全系数
  安全系数 = 1.5（CCS/DNV要求）
  τ_allowable = 轴材料剪切疲劳极限（锻钢约160~200 MPa）

典型校核示例：
  d = 120mm，材料45钢（τ_allowable=180MPa）
  静扭矩 T_static = 1.91 kN·m → τ_static = 11.2 MPa
  动态扭矩幅值 T_dynamic = 0.5 kN·m → τ_dynamic = 2.9 MPa
  合计：14.1 MPa << 180/1.5 = 120 MPa  ✅
```

### Step 5：共振处理措施

```
措施1（首选）：调整联轴器刚度，改变固有频率
  降低 k_coupling → 降低 f_n1
  升高 k_coupling → 升高 f_n1
  调节范围：通常±30~50%

措施2：增加扭振减震器
  橡胶型：阻尼比提升至0.1~0.15，共振幅值降低50%以上
  硅油型：阻尼比可达0.2~0.25，适用于无法错开的共振

措施3：设置禁速区+快速穿越（控制措施）
  适用条件：共振点在非常规工作转速处（偶尔经过）
  穿越要求：加速度 ≥ 5rpm/s

⚠️ 禁止："不做分析，直接运行试试看"
   共振未处理的后果：联轴器橡胶元件疲劳断裂（通常1~4周）
                     轴颈疲劳裂纹（通常3~6个月）
                     严重时轴断裂→螺旋桨飞出（灾难性）
```

**典型案例完整复盘：**

```
案例：某内河推船300rpm附近持续振动，联轴器橡胶件一月损坏一次
      累计维修成本：约6万元/年，停航时间约10天/年

诊断：
  振动主峰频率 = 20Hz（300rpm时）
  建立集总参数模型：f_n1 = 19.2 Hz（含附水质量修正）
  4叶桨300rpm时：f_激励 = 4×300/60 = 20 Hz ≈ f_n1 → 共振确认

整改方案：
  A：换联轴器（刚度降低30%）→ f_n1降至15Hz → 共振点移至225rpm（低速工况）
     成本：1.2万元
  B：控制禁速区（290~310rpm快速穿越）
     成本：0.2万元
  C = A + B（组合方案，最可靠），总成本：1.4万元

整改结果：
  振动消除，联轴器寿命恢复正常（>2年）
  年维修费+停航损失降低约20万元
  投资回收期：1.4万元 / 20万元 ≈ 0.07年（约25天回本）
```

## 5.3 轴系校中分析（四步法）

### Step 1：冷态校中

```
测量工具：激光对中仪（精度0.001mm，推荐使用）

限值要求：
  联轴器径向偏差 ≤ 0.05mm/100mm（跨距）
  联轴器角向偏差 ≤ 0.05mm/100mm
  轴承座高度差   ≤ 0.10mm

典型测量记录表：
  测量位置          读数(mm)    限值(mm)    合格?
  ─────────────────────────────────────────────
  联轴器径向偏差     0.02        ≤0.05       ✅
  联轴器角向偏差     0.03        ≤0.05       ✅
  轴承1高度偏差     +0.05       ≤0.10       ✅
  轴承2高度偏差     -0.03       ≤0.10       ✅
```

### Step 2：热态补偿预留量

```
计算公式：
  ΔH = α × H_center × ΔT

  α = 热膨胀系数 = 11.7×10⁻⁶/°C（碳钢）
  H_center = 电机轴中心高度（从电机底脚到轴中心距离，mm）
  ΔT = 运行温度 - 环境温度（典型值40~80°C）

示例：H=500mm，ΔT=60°C
  ΔH = 11.7×10⁻⁶ × 500 × 60 = 0.35mm
  → 安装时电机侧预留0.35mm"冷态低位"
  → 热机后电机上升0.35mm → 同轴恢复 ✅

⚠️ 反例：未做热态补偿计算，冷态校中合格
   热机后电机侧上升0.3~0.5mm → 联轴器角向偏差超限
   → 联轴器加速磨损，轴承载荷分布不均
```

### Step 3：轴承载荷分布（三弯矩方程法）

```
三弯矩方程（连续梁模型）：
  M_i-1 × L_i-1 + 2M_i × (L_i-1 + L_i) + M_i+1 × L_i
  = -6EI × (δ_i-1/L_i-1 + δ_i+1/L_i)

  δ_i = 轴承i的垂直位移（由热变形/安装误差引起）

载荷限值要求：
  每个轴承动态载荷 ≤ 额定载荷 × 0.8（20%安全裕度）
  最轻载荷轴承 ≥ 轴段重量 × 10%（防止轴飞边）
  推力轴承：正车推力+倒车推力均需独立校核
```

### Step 4：海船基座变形分析

```
海船特有问题：波浪弯曲 → 船体梁弯曲 → 轴系基座位移 → 对中改变

分析方法：
  船体有限元分析（FEA）→ 典型波浪工况下基座节点位移
  将基座位移作为轴系分析的边界条件

典型结果：在波浪中，轴承载荷变化范围可达 ±30% 静态值

适用范围：
  内河船：通常不需要（内河无大浪）
  近海船：可做简化分析
  海船（跨洋）：必须做，CCS/DNV审图强制要求
```

## 5.5 推力轴承 L10 寿命详细计算

```
ISO 281 修正寿命计算：

基本额定动载荷寿命：
  L10 = (C/P)^p × 10⁶ 转
  C = 轴承基本额定动载荷（kN，从厂家样本查取）
  P = 当量动载荷（kN）
  p = 3（球轴承）/ 10/3（滚子轴承）

当量动载荷计算（推力轴承）：
  P = X × F_r + Y × F_a
  F_r = 径向载荷（轴系重量分量，kN）
  F_a = 轴向载荷（螺旋桨推力，kN）
  X、Y = 径向和轴向系数（从厂家样本查取）

折算为小时寿命：
  L10_h = L10 × 10⁶ / (60 × n)
  n = 额定转速（rpm）

任务剖面加权计算（Miner 法则）：
  D = Σ(n_i / N_i)
  N_i = 工况i下对应的理论疲劳寿命（转数）
  D < 1 → 寿命满足要求

示例（2000kW，300rpm，轴向推力F_a=80kN，C=1200kN，X=0.4，Y=1.2，F_r=15kN）：
  P = 0.4×15 + 1.2×80 = 6 + 96 = 102 kN
  L10 = (1200/102)^(10/3) × 10⁶ 转
      = 11.76^(3.33) × 10⁶
      ≈ 2210 × 10⁶ 转
  L10_h = 2210×10⁶ / (60×300) = 122,800 h >> 40,000h ✅

  任务剖面加权（经济航速75%工况占60%时长）：
  D = (60%×年运营时间×300rpm×60) / N(工况75%)
    + (10%×年运营时间×300rpm×60) / N(工况100%)
    + ...
  D << 1 → 设计寿命满足25年要求 ✅

CCS 最低要求：B10 ≥ 40000h（相当于约4.5年×8760h/年）
推荐目标：B10 ≥ 100000h（含任务剖面加权，覆盖20年船龄）
```

## 5.6 传动系统 CDR 检查单

```
螺旋桨-电机匹配：
  □ T-n匹配图完成，电机曲线包住所有工况点
  □ 倒车工况校核完成（四象限运行能力确认）
  □ 堵转保护：软件过流保护+硬件扭矩限制器双重方案设计完成

扭振分析：
```

## 5.7 本章小结与关键输出物

**本章核心要点（四条）：**

1. **T-n 匹配是传动设计第一步，不可跳过**
   匹配图中发现的转矩裕度不足，在此阶段修改电机选型成本为零；CDR后发现需要换电机，成本可达数十万元。

2. **扭振分析是船级社强制项，附水质量不可忽略**
   忽略螺旋桨附水质量→固有频率偏高10~15%→禁速区位置错误→在真实共振区运行→联轴器断裂。

3. **热态补偿是容易遗忘的细节，后果严重**
   0.35mm冷态补偿量看似微小，忽略后导致联轴器角向偏差超限，联轴器和轴承均加速磨损。

4. **L10寿命按任务剖面加权计算，非单工况**
   按额定工况算出寿命足够，按任务剖面加权计算结果可能差异很大（不同工况推力差异巨大）。

**关键输出物：**
