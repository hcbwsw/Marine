# 第四部分导读：规范与合规

> 本部分回答：**我的设计要满足哪些规范？如何一次通过船级社审图，避免返工？**

## 学习目标

读完本部分，你应该能够：

1. 列出项目适用的 CCS/DNV/IEC 规范清单
2. 建立规范条款到设计参数的合规映射矩阵
3. 说清楚内河船与海船的设计差异
4. 预判并规避新能源电推的七大特有技术问题

## 工程要点

- **合规不是事后盖章**：它是贯穿整个研发生命周期的设计约束
- **热失控不蔓延**是最严苛的试验：很多电池系统在此项被卡
- **内河与海船逻辑不同**：内河重成本和快充，海船重冗余和可靠性
