|----------|------|------|
| Rules for Classification of Ships | DNV-RU-SHIP | 船舶入级规范 |

## 2. 电池安全要求（Pt.3 Ch.2 Sec.3）

### 2.1 一般要求

- 电池系统需通过 DNV Type Approval
- 电池系统应设计为故障安全型（fail-safe）
- 单个电芯热失控不应导致火灾或爆炸蔓延
- 电池系统应进行 FMEA（故障模式与影响分析）

### 2.2 电池舱要求

| 项目 | DNV 要求 | CCS 要求 | 对比 |
|------|----------|----------|------|
| 防火分隔 | A-60 | A-60 | 一致 |
| 通风换气 | ≥ 20 次/h | ≥ 20 次/h | 一致 |
| 温度监测密度 | 每 2m² 至少 1 个 | 每模组至少 2 个 | DNV 更严格 |
| 泄爆设计 | 强制要求 | 强制要求 | 一致 |
| 灭火系统 | 固定式 | 固定式 | 一致 |
| 绝缘监测 | 强制 | 强制 | 一致 |
| 紧急切断 | 舱外可达 | 舱外可达 | 一致 |

### 2.3 热失控要求

**DNV 热失控试验要求：**
- 触发方式：加热、针刺或过充
- 判定标准：
  - 相邻电芯温度 < 100°C
  - 无火焰蔓延到相邻模组
  - 电池舱结构完整性保持
  - 通风系统正常运行
- 需提交热失控仿真报告（CFD 分析）

📌 **DNV 比 CCS 更严格的地方**：要求提交热失控仿真报告，且温度监测密度更高。

### 2.4 BMS 要求

- BMS 需独立于其他控制系统
- BMS 故障不应导致电池系统失控
- BMS 应具备冗余通信通道
- 需进行 BMS 功能安全评估（IEC 61508 SIL 2）

## 3. 与 CCS 规范对比

### 3.1 认证流程对比

| 项目 | CCS | DNV |
|------|-----|-----|
| 型式认可 | 需要 | Type Approval |
| 热失控试验 | 需要 | 需要 + 仿真报告 |
| FMEA | 建议 | 强制 |
| 功能安全 | IEC 61508 | IEC 61508 SIL 2 |
| 环境试验 | IEC 60068 | DNV-CG-0339 |

### 3.2 主要差异

| 项目 | CCS | DNV |
|------|-----|-----|
| 规范体系 | 国内规范，中文 | 国际规范，英文 |
| 适用范围 | 国内船舶为主 | 国际航行船舶 |
| 审查周期 | 相对较短 | 相对较长 |
| 费用 | 较低 | 较高 |
| 国际认可度 | 国内认可 | 全球认可 |

💡 **选型建议**：国内内河船舶选 CCS 即可；出口船舶或国际航线需 DNV 或其他国际船级社认证。

## 4. 实操注意事项

### 4.1 DNV 认证常见问题

- 技术文件需英文版本
- 热失控仿真需使用认可的 CFD 软件
- FMEA 报告需由有资质的工程师签署
- 工厂审查包括生产过程和质量体系

### 4.2 认证周期

- 电池系统 Type Approval：6-12 个月
- 船舶入级检验：根据船型和复杂度，3-6 个月
- 建议提前规划，与电池供应商同步推进
