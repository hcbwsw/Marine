

# 第29章 手册总结与最终建议

## 从技术设计到商业成功的完整闭环

本手册涵盖了船舶新能源电推系统从**概念设计→详细工程→测试验证→商业化**的完整生命周期。这不是一本单纯的技术书，而是一份**工程管理与商业决策的综合指南**。

---

## 23.1 核心要点回顾

### 三个层次的系统思维

#### **第一层：技术深度(第7章-第18章)**
从总体框架、需求工程、系统架构，到专项系统设计（冷却、传动、控制）、新能源特殊问题，最后进行RAMS可靠性评估与合规映射。

**关键领悟**：
- ✓ 不要急于选择技术方案，先充分理解需求与约束
- ✓ 系统架构设计是所有后续工作的基础，投入充足时间值得
- ✓ 冷却系统往往是被忽视的薄弱环节，需要特别关注
- ✓ 新能源不是简单替换柴油，而是全新的系统思维

#### **第二层：工程执行(第19章-附录B)**
验证与确认体系(V&V)、新能源特有技术问题、内河/海船差异化设计、规范与合规、90天落地执行、工具链与学习路径。

**关键领悟**：
- ✓ V&V贯穿全生命周期，不是交付前才做
- ✓ 新能源电推的电池热管理、可靠性、成本是三大挑战
- ✓ 内河与海船需要差异化设计，不能一刀切
- ✓ 合规映射工作常被低估，实际占工作量的20-30%
- ✓ 90天落地执行计划需要每周滚动更新，不是一成不变

#### **第三层：商业化(第13章-第28章)**
从船舶与陆地测试体系、商业化与产品化、到技术路线图与前沿趋势。

**关键领悟**：
- ✓ 技术验证(FAT/SAT)是确保产品可靠性的最后防线
- ✓ 全寿命数据追踪系统是建立品牌信誉的基础
- ✓ 成本优化是持续的工作，不是一劳永逸
- ✓ 供应链管理与技术创新同样重要

### 五个必须避免的陷阱

```
陷阱1：技术导向而非需求导向
✗ 错误做法：选择最新/最酷的技术，忽视实际需求
✓ 正确做法：从客户需求出发(续航/成本/可靠性)，反推技术选择

陷阱2：忽视冷却系统的重要性
✗ 错误做法：冷却系统交给第三方外包，不参与设计评审
✓ 正确做法：冷却系统与电机控制同级重要，需要深度参与

陷阱3：过度设计导致成本失控
✗ 错误做法：为了"安全"，加入各种冗余，成本翻倍
✓ 正确做法：基于FMEA与风险评估，必要的冗余才加入

陷阱4：V&V流于形式
✗ 错误做法：对着设计文件验证，现场发现问题
✓ 正确做法：逐项验证，陆地+海试双层把关

陷阱5：忽视供应链与成本的战略性
✗ 错误做法：只关注技术指标，成本由采购部门决定
✓ 正确做法：在设计阶段就考虑成本与供应链，并参与采购

```

### 四个决策框架

#### **决策框架1：技术路线选择**

```
问题：我应该选择BEV/HEV/FCHEV中的哪一个?

决策流程：
  第1步：定义客户需求
    ├─ 续航能力(nm): ?
    ├─ 日返港要求(y/n): ?
    ├─ 成本敏感度(高/中/低): ?
    └─ 环保要求(等级): ?
  
  第2步：成熟度评估(当前2024)
    ├─ BEV: ★★★★★ 成熟，可商用
    ├─ HEV: ★★★★☆ 相对成熟，可商用
    ├─ FCHEV: ★★☆☆☆ 概念期，不建议
    └─ 其他: 预留，持续观察
  
  第3步：成本评估(2024)
    ├─ BEV: 250万(最低)
    ├─ HEV: 280万(+12%)
    ├─ FCHEV: 400万+(成本未知)
    └─ 传统柴油: 150万(基准)
  
  第4步：应用场景匹配
    内河日返(<100nm)  → BEV
    沿海日返(<300nm)  → HEV(过渡) OR BEV(长期)
    远洋(>1000nm)     → 等待 FCHEV 2030+ OR 保持柴油
    港口作业(固定)    → BEV(充电便利)

结论：
  ├─ 若续航<400nm，日返: 选择BEV ✓
  ├─ 若续航400-800nm: 选择HEV(2026-2028) ✓
  ├─ 若续航>1000nm: 等待 FCHEV ✗(暂时)
```

#### **决策框架2：自主研发 vs 采购集成**

```
问题：电机/电池/控制系统应该自主开发还是采购?

评估维度：
                自主研发           采购集成
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
技术难度        ★★★★★           ★★
投入成本        高(5000万+)        低(1000万)
开发周期        长(3-5年)          短(12-18个月)
知识产权        完全自主            依赖供应商
技术创新        高(可领先)         中(跟随)
成本竞争力      长期优势           初期有利
供应链风险      自承担             供应商承担
组织能力需求    高(需要人才)      中(集成能力)

决策建议：
  核心系统(电推控制算法): 自主研发 ✓
    理由：差异化竞争力，不可依赖第三方
  
  通用件(电机/电池):     采购+集成 ✓
    理由：市场成熟，无需重复开发
  
  中间件(BMS/冷却控制):   自主+代工 ✓
    理由：部分自主研发，代工生产

最终模型：
  ├─ 自主开发占比: 40-50% (算法+电控)
  ├─ 采购占比: 30-40% (电机/电池/变频器)
  └─ 集成占比: 20-30% (系统整合)
```

#### **决策框架3：市场进入策略**

```
问题：如何从0开始进入新能源船舶推进市场?

三个阶段策略：

Stage 1: 样船试水(0-12个月)
  目标：获得1-2艘样船订单，验证技术与市场
  策略：
    ├─ 聚焦细分市场(港口/内河二级城市)
    ├─ 与头部客户合作(获得背书)
    ├─ 不追求利润，专注可靠性验证
    ├─ 成本可承受亏本(每艘-50万)
    └─ 目标：完成2艘样船，积累案例
  
  KPI:
    ├─ 签订订单数: ≥1艘
    ├─ 合同金额: ≥300万
    ├─ 客户满意度: ≥9/10
    └─ 故障率: <2次/月

Stage 2: 规模扩张(13-36个月)
  目标：从样船转向批量，建立市场地位
  策略：
    ├─ 利用样船案例获得更多订单
    ├─ 目标客户扩展(不同地区/船队)
    ├─ 成本优化，毛利提升至10-12%
    ├─ 建立销售渠道(区域代理)
    └─ 目标：交付30-50艘，市场知名度top 3
  
  KPI:
    ├─ 年交付量: 10-20艘
    ├─ 毛利率: 10-12%
    ├─ 市场占有率: 5-8%
    └─ 品牌认知: "行业新兴品牌"

Stage 3: 市场领导(37个月+)
  目标：成为行业标杆，国际竞争力
  策略：
    ├─ 国际扩展(东南亚/欧洲)
    ├─ 产品升级(第二代/第三代)
    ├─ 成本竞争力与差异化并重
    ├─ 兼并收购或IPO融资
    └─ 目标：全球top 3品牌
  
  KPI:
    ├─ 年交付量: 100+艘
    ├─ 毛利率: 15-20%
    ├─ 市场占有率: 15-20%
    └─ 全球品牌知名度
```

#### **决策框架4：融资与投资回报**

```
问题：投入多少钱才能做成新能源电推系统?

投资规模评估：

规模              初期研发投入     3年累计投入     人员规模
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
小型创业企业      1000-2000万      3000-5000万    20-30人
中型技术公司      3000-5000万      8000-15000万   50-80人
大型传统企业      5000-10000万     20000-30000万  100+人

回报周期与模式：

模式1：VC融资+IPO退出(创业企业)
  投资人预期：
    ├─ 3-5年内完成A/B轮融资(融资3-10亿)
    ├─ 第5年达成IPO(上市估值20-50亿)
    └─ VC回报倍数: 10-30倍
  
  企业需要：
    ├─ 快速增长(年增长>100%)
    ├─ 建立技术壁垒(专利/品牌)
    └─ 找到战略退出(IPO/被收购)

模式2：内部融资+业务自给(传统企业改造)
  投资人预期：
    ├─ 3-5年内新业务贡献>20%收入
    ├─ 新业务毛利>15%，参与集团利润
    └─ 战略意义>财务回报
  
  企业需要：
    ├─ 集团资金支持(不依赖外融)
    ├─ 耐心(不急于ROI)
    └─ 长期承诺(5-10年)

模式3：产业联动+生态合作(中等企业)
  投资人预期：
    ├─ 参与产业链协作(电池/造船/运营)
    ├─ 通过合作快速占领市场
    └─ 3-5年达成并购整合
  
  企业需要：
    ├─ 开放心态(接纳合作)
    ├─ 补足短板(技术/渠道/资金)
    └─ 明确的赛道(不要什么都做)

投资回报时间表示例(中等企业)：

Year 1-2: 亏本(投入>收入)
  累计亏损: -5000万

Year 3: 收支平衡
  收入: 8000万，成本: 8000万，利润: 0

Year 4: 开始盈利
  收入: 20000万，毛利: 12%，税前利润: 2400万

Year 5: 利润倍增
  收入: 40000万，毛利: 15%，税前利润: 6000万

累计5年回报：
  └─ 5年累计利润: 1.5亿元(若Year 1-2无亏损)
  └─ IRR(内部收益率): 25-35%
  └─ 对标: 银行收益3%，股市收益8%，新能源企业IRR>20%为优秀

## 23.2 按岗位的实用建议

### 对技术总监(CTO)的建议

```
【你面临的核心问题】
  "如何在3年内从0完成产品从设计→量产，并赶上市场窗口期？"

【我的建议】

1. 第一个月(月度1)：架构设计，一次做对
   行动：
     □ 聚集核心设计团队(5-8人)
     □ 输出系统架构设计文档(第9章)
     □ 完成关键技术选择(BEV vs HEV)
     □ 签署NDA，保护知识产权
   
   为什么重要：架构错了，后续所有工作都是浪费

2. 第2-3个月：快速原型与陆地测试
   行动：
     □ 购置台架测功机(或租赁)
     □ 集成关键部件(电机+变频+电池+控制器)
     □ 跑通基本功能(转速+温度+功率)
     □ 50项FAT测试，力争通过>80%
   
   为什么重要：及早发现集成问题，不要等到海试

3. 第4-6个月：第一艘样船海试
   行动：
     □ 与造船厂合作，完成船舶集成
     □ 执行SAT验证计划(2-4周)
     □ 收集真实运营数据(1个月)
     □ 汇总故障与改进建议
   
   为什么重要：海试是最真实的测试，给后续产品迭代的方向

4. 第7-12个月：产品迭代与成本优化
   行动：
     □ 基于海试反馈，修改设计(BOM/控制算法)
     □ 启动第二艘样船(不同工况)
     □ 降低成本目标达成15-20%
     □ 获得第一个商业订单
   
   为什么重要：不要停留在研发，转向产品化与商业化

5. 持续工作(Year 2-3)：
   □ 建立可靠性增长体系(数据驱动)
   □ 平衡创新与稳定(20% 新功能，80% 可靠性改进)
   □ 招聘与培养团队(从5→20人)
   □ 专利申请与技术积累
   □ 参加行业会议，建立业界声望

【最关键的KPI】
  ├─ Year 1: FAT通过率>90%，第一艘样船SAT满意度>9/10
  ├─ Year 2: MTBF>2000h，毛利>0%，获得3-5个商业订单
  └─ Year 3: MTBF>4000h，毛利>15%，交付20-30艘

【你必须避免的错误】
  ✗ 过度设计(导致成本失控)
  ✗ 完美主义(无限延期交付)
  ✗ 忽视可靠性(早期只关注功能)
  ✗ 团队不稳定(核心人员流失)
  ✗ 技术孤岛(不与商业/供应链沟通)
```

### 对采购总监的建议

```
【你面临的核心问题】
  "如何在有限预算内，采购到稳定的关键零部件，同时降低成本？"

【我的建议】

1. 第一个月：供应商识别与评估
   行动：
     □ 逐一调研电机/电池/变频器供应商
     □ 邀请TOP 3供应商参与设计评审
     □ 收集报价与技术资料
     □ 建立供应商评分卡
   
   目标供应商(参考2024)：
     ├─ 电池: CATL/BYD/国轩/孚能
     ├─ 电机: ABB/西门子/三菱/中国电机厂
     ├─ 变频器: 西门子/英飞凌/华为(新兴)
     └─ 控制系统: 自研 + 与大学合作

2. 第2-3个月：合同谈判与定价
   行动：
     □ 与A级供应商（电池/电机）谈3-5年框架合同
     □ 定价策略：基于3年预测量(与成本目标关联)
     □ 设置volume discount(年度量超50套，降价5-10%)
     □ 技术升级条款(电池容量提升，价格同步调整)
   
   定价原则：
     ├─ Year 1: 基准价(无折扣)
     ├─ Year 2: 体积折扣5%
     └─ Year 3: 体积折扣10% + 技术升级分享

### 对销售总监的建议

```
【你面临的核心问题】
  "如何在竞争激烈的市场中，从零开始获得第一个商业订单？"

【我的建议】

1. 第一个月：客户洞察与市场细分
   行动：
     □ 调研top 100潜在客户(航运公司/港口/政府部门)
     □ 分类(A类=高价值，B类=中等，C类=低)
     □ 建立客户档案(需求/预算/决策周期)
     □ 识别关键决策人
   
   客户分类示例：
     ├─ A类：国企航运集团(中国内河、长江)
     ├─ B类：地方港口与工程公司
     └─ C类：小型民营船队

2. 第2-3个月：差异化定位与价值主张
   行动：
     □ 定位：对标竞争对手(国外品牌vs国内初创)
     □ 价值主张(我为什么比别人好？)
     □ 定价策略(溢价vs平价)
     □ 营销物料准备
   
   建议定位(2024-2025)：
     ├─ 差异化: "成本低+本地服务"vs"国外技术"
     ├─ 定价: -10% vs国际品牌
     ├─ 承诺: 5年质保+24h技术支持
     └─ 目标客户: 国内二三线港口(成本敏感)

3. 第4-6个月：样船营销与案例积累
   行动：
     □ 与首批样船客户建立紧密合作
     □ 邀请潜在客户参观样船(现场演示)
     □ 收集客户反馈与建议
     □ 总结案例故事(成本节省量化)
   
   案例故事模板：
     "××港口选择我们的产品，相比国外品牌：
      - 成本节省: 30万(12%)
      - 续航提升: 15%(通过本地化调优)
      - 服务响应: 4h(vs国外48h)
      → 客户满意度: 9.5/10"

4. Year 2: 销售渠道与客户开发加速
   行动：
     □ 建立销售代理网络(5-10个地区)

【最关键的KPI】

### 对质量总监的建议

```
【你面临的核心问题】
  "如何确保产品可靠性，同时不过度设计导致成本失控？"

【我的建议】

1. 第一个月：建立质量体系框架
   行动：
     □ 制定QMS手册(ISO 9001基础)
     □ 定义质量目标(MTBF > 2000h)
     □ 建立检验点(进料/制程/出货)
     □ 培训质量小组
   
   质量政策示例：
     "零缺陷是目标，但通过预防而不是检验实现"

2. 第2-3个月：FMEA与设计评审
   行动：
     □ 与设计团队执行系统FMEA
     □ 识别高风险项(RPN > 100)
     □ 对高风险项制定改进措施
     □ 跟踪改进验证
   
   FMEA工作分配：
     ├─ 电机系统: S=4, O=3, D=4 → RPN=48(中等)
     ├─ 电池热管理: S=5, O=2, D=3 → RPN=30(中等)
     ├─ 控制系统: S=5, O=2, D=2 → RPN=20(低)
     └─ 冷却系统泄漏: S=4, O=3, D=5 → RPN=60(需改进)

3. 第4-6个月：FAT与来料检验标准
   行动：
     □ 编制50项FAT测试用例
     □ 建立来料检验标准(AQL 2.5)
     □ 建立制程检验规范(IPQC检查点)
     □ 设计检验记录表格
   
   关键检验点(最容易出现的问题)：
     ├─ 电池: SOC准度、内阻、温度特性
     ├─ 电机: 绝缘、转矩、噪音
     ├─ 冷却: 流量、温度响应、无泄漏
     └─ 控制系统: 通讯可靠性、故障诊断准确性

4. Year 2: 可靠性增长与持续改进
   行动：
     □ 建立故障追踪系统(FRACAS)
     □ 每月分析故障数据，RCA分析
     □ 对设计/工艺进行改进
     □ 建立零缺陷文化(全员参与)
   
   可靠性增长曲线(示例)：
     ├─ Year 1: MTBF = 800h (初期)
     ├─ Year 1 Q4: MTBF = 1500h (+88%)
     ├─ Year 2 Q2: MTBF = 2500h (+67%)
     └─ Year 2 Q4: MTBF = 4000h (+60%)
   
   这是正常的可靠性增长曲线，每个季度改进10-30%

5. Year 3: 卓越运营
   行动：
     □ 整体质量体系通过ISO 9001认证
     □ 供应商质量管理成熟(SQE体系)
     □ 建立预测维护系统(基于大数据)
     □ 行业标准引领者

【最关键的KPI】
  ├─ MTBF: Year1:1500h → Year2:4000h → Year3:8000h
  ├─ 首通过率(First Pass Yield): >90%
  ├─ 客户满意度: >9/10
  ├─ 召回率: <0.1%(重大质量问题)
  └─ 质量成本: <总成本3%

【你必须避免的错误】
  ✗ 过度检验(100% 检验无法保证可靠性)
  ✗ 只关注故障数，不做RCA(问题反复出现)
  ✗ 质量部门与设计部门对立(应该合作)
  ✗ 忽视供应商质量(80%问题来自供应商)
  ✗ 不投入预防(等到出现问题才行动)
```

## 23.3 常见问题解答(FAQ)

```
Q1: 新能源电推系统开发需要多长时间？
A:  取决于复杂度和企业基础：
    ├─ 完全从零开始: 3-5年(包括量产)
    ├─ 有造船经验，新增新能源: 2-3年
    ├─ 有新能源经验，新增船舶: 2-3年
    └─ 两方都有经验: 1-2年
    
    关键是前12个月(需求+设计+原型)，这个做对了，后续加速。

Q2: 第一批产品应该生产多少艘？
A:  建议：5-10艘样船
    ├─ 不要太多: 容易固化问题，无法快速迭代
    ├─ 不要太少: 无法积累经验，样本量太小
    ├─ 5-10艘: 足以验证可靠性，同时可快速改进
    └─ 每艘应用不同工况(港口/内河/沿海)，获取全面数据

Q3: 成本目标应该如何制定？
A:  建议：相对价值法
    ├─ 对标竞争对手价格(国外品牌、国内初创)
    ├─ 确定目标毛利率(初期0-5%，成熟15-20%)
    ├─ 反推成本目标: 成本 = 价格 × (1 - 毛利%)
    
    示例(300kW混合动力)：
      竞争对手价格: 280万
      目标定价: 250万(-10%降价竞争)
      目标毛利: 10%
      → 成本目标: 250万 × 90% = 225万
      (当前BOM 180万，有35万空间用于制造/服务/管理)

Q4: 如何选择第一个样船客户？
A:  建议标准：3个要素
    ├─ 敢于创新: 愿意尝试新产品(大企业较好)
    ├─ 支持配合: 积极反馈数据(产业链客户理想)
    └─ 有耐心: 可接受初期问题(不会鸡蛋挑骨头)
    
    避免的客户：
      ✗ 苛刻的客户(初期样品无法满足)
      ✗ 不愿合作的客户(数据收集困难)
      ✗ 财务困难的(可能无法支付)

Q5: 何时启动国际化？
A:  建议时间表：
    ├─ Year 1-2: 专注国内，建立基础
    ├─ Year 2-3: 评估与试点(东南亚1-2个订单)
    ├─ Year 3+: 全面国际化(欧美/东南亚并行)
    
    国际化前提条件：
      ✓ 国内订单>50艘，口碑稳定
      ✓ 产品通过DNV/GL等国际船级社认证
      ✓ 有国际销售与服务能力
      ✗ 不要为了融资而急于国际化

Q6: 如何平衡技术创新与可靠性？
A:  建议比例：20-80法则
    ├─ 20% 新功能/创新(新型传感器、AI诊断等)
    ├─ 80% 可靠性改进(提升MTBF、降低成本)
    
    为什么？
      初期客户需要的是稳定产品(能用)
      不是功能丰富但故障频繁的产品(花哨)
    
    创新的时机：
      Year 1-2: 专注可靠性(60% 新功能:40% 可靠性)
      Year 2-3: 平衡创新(40% 新功能:60% 可靠性)
      Year 3+: 积极创新(50% 新功能:50% 可靠性)

Q7: 人才最困难的是什么？
A:  三个层次的人才短缺：
    ├─ 技术人才: 懂新能源 + 懂船舶的(最稀缺)
    ├─ 管理人才: 既懂工程又懂商业的
    └─ 执行人才: 愿意加班、能吃苦的工人
    
    解决方案：
      ✓ 技术人才: 与大学建立实习基地，内部培养
      ✓ 管理人才: 从传统行业挖角(造船/汽车)
      ✓ 执行人才: 提供有竞争力的薪资+股权激励
      ✓ 文化激励: 让员工相信"这是改变世界的工作"

Q8: 什么时候可以考虑IPO或融资？
A:  建议阶段：
    ├─ A轮融资(种子期): Year 1, 融资1-3亿, 目的:产品验证
    ├─ B轮融资(成长期): Year 2, 融资5-10亿, 目的:规模化
    ├─ Pre-IPO(成熟期): Year 3-4, 融资10-20亿, 目的:国际化
    └─ IPO(上市): Year 4-5, 融资20-50亿, 目的:资本运作
    
    IPO前的关键指标：
      ✓ 年收入>10亿
      ✓ 毛利率>15%
      ✓ MTBF>5000h(产品成熟)
      ✓ 市场占有率>10%
      ✓ 国际认证完整

Q9: 需要建立自己的生产线吗？
A:  建议：初期不需要，代工模式优先
    ├─ Year 1-2: 与代工厂合作(代工方承担产能风险)
    ├─ Year 2-3: 部分自建(组装+质检在自己工厂)
    ├─ Year 3+: 核心工序自建(电池management system、控制器)
    
    优势：
      ✓ 初期投资少(不需要几亿建厂)
      ✓ 灵活性强(产能调整容易)
      ✓ 聚焦核心(专注设计和销售)
    
    劣势：
      ✗ 质量控制需要严格(代工厂质量水平参差不齐)
      ✗ 成本控制困难(代工厂利润率高)
      ✗ 产品保密风险(工艺可能泄露)

Q10: 遇到重大技术问题怎么办？
A:  应急预案(3步)：
    ├─ 第1步(24小时): 停止生产/交付，组建RCA团队
    ├─ 第2步(1周): 完成问题诊断，提出改进措施
    ├─ 第3步(2周): 验证改进有效，通知客户解决方案
    
    对客户的态度：
      ✓ 主动沟通(不要等客户发现)
      ✓ 承担责任(不要甩锅)
      ✓ 快速解决(用行动赢得信任)
      ✓ 后续跟踪(问题解决后1个月再电话回访)
    
    历史案例：
      某新品上市后发现电池过热问题
      快速RCA + 改进，获得客户谅解
      后续订单反而更多(因为快速响应被评价)
```

## 23.5 最后的寄语

### 给创业者的话

```
如果你正在考虑进入新能源电推船舶领域：

【先问自己3个问题】

问题1：为什么是船舶推进，不是其他领域？
  好答案：
    ✓ "我对航运业有深入理解，看到了明确的市场需求"
    ✓ "我的技术优势天然适合这个应用"
    ✓ "我有行业人脉和客户基础"
  
  坏答案：
    ✗ "因为有补贴"
    ✗ "看别人做这个赚钱"
    ✗ "我在新能源领域，随便应用到船舶"

问题2：你愿意花3-5年才看到真正的收益吗？
  这不是一个快速赚钱的生意：
    ├─ Year 1-2：亏损5000万以上
    ├─ Year 3：开始微利
    ├─ Year 4：真正盈利
    └─ Year 5+：收获期
  
  如果你的投资方要求2年见效，放弃吧。

问题3：你能承受失败吗？
  失败的可能性包括：
    ├─ 技术方向选错(燃料电池延期导致浪费投入)
    ├─ 市场需求不如预期(补贴取消导致销售停滞)
    ├─ 竞争对手抢占市场(国际巨头进入)
    ├─ 供应链中断(关键部件缺货)
    └─ 产品可靠性问题(需要大规模改进)
  
  如果答案是"不能"，那就不适合创业。

【成功的条件】

最关键的三个条件(按重要性排序)：
  1. 人才(聪明的工程师 + 有执行力的管理层)
  2. 资金(充足的资金支持熬过前3年)
  3. 客户(有前瞻眼光的早期客户支持)
  
  其他都是次要的(技术会改进，产品会迭代)

【我的建议】

  ✓ 别想着成为全球第一，先成为某个细分领域的第一
    (港口作业? 内河日返? 岛屿供应?)
  
  ✓ 别想着做完美产品，先做"足够好的产品"
