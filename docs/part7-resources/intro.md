# 第七部分导读：工具与资源

> 本部分是工具箱：**计算器、案例、参考资料，随取随用。**

## 内容地图

| 附录 | 内容 | 使用场景 |
|------|------|----------|
| 附录A 交互计算工具 | 电池容量/推进功率/续航/效率 | 方案估算、快速校核 |
| 附录B 工具链与学习路径 | 软件工具、学习资源 | 搭建工作环境 |
| 附录C 案例库 | 实际项目案例 | 设计参考、对标 |
| 附录D 参考资料 | 术语表/规范清单/书目/链接 | 查术语、找标准 |
| 附录E 规范解读笔记 | 规范条款的实操解读 | 审图问题速查 |

## 使用建议

- **计算器有默认值**：打开就能算，默认值是内河客船典型参数
- **案例库持续填充**：每完成一个项目，按模板整理一个案例
- **规范笔记随手记**：审图遇到的问题和解读，记到附录E，下次直接查
