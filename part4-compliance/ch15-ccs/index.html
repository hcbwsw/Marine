    2. 电池动力相关条款¶

    2.1 电池系统设计¶

    一般要求： - 电池系统应通过型式认可 - 电池组额定能量应满足船舶最大工况续航要求 - 电池系统应设置冗余，至少 N+1 配置 - 电池舱应独立设置，不应与客舱、机舱混用

    电压等级： - 直流系统电压不超过 1000 V - 超过 60V 属于高压系统，需绝缘监测

    2.2 BMS 要求¶

    • 每个电池簇应配备独立的 BMS
    • BMS 应实现：电压监测、温度监测、电流监测、SOC/SOH 估算、均衡管理、保护功能
    • BMS 应具备自诊断功能
    • BMS 应与 EMS 通信，实时上传状态数据
    • BMS 故障不应导致电池系统失控

    2.3 电池舱安全¶

    结构要求： - 电池舱与相邻舱室之间应采用 A-60 级防火分隔 - 电池舱不应直接位于客舱下方 - 电池舱应设置独立通风系统，换气次数 ≥ 20 次/小时 - 通风口不应朝向客舱、救生设备等区域

    监测与报警： - 每个电池模组应设置温度传感器（至少 2 个） - 电池舱应设置感烟探测器和可燃气体探测器 - 报警信号应传送至驾驶室和机舱集控室

    灭火系统： - 电池舱应设置固定式灭火系统 - 推荐介质：七氟丙烷、细水雾 - 灭火系统应能在电池舱外触发

    泄爆设计： - 电池舱应设置泄压装置 - 泄压方向应朝向安全区域 - 泄压面积应按电池舱容积和电池能量计算

    2.4 充电系统¶

    • 充电装置应通过型式认可
    • 充电过程中电池舱应保持通风
    • 充电连接器应具有防误插和防触电设计
    • 快充系统需与岸方协调供电容量
    • 充电系统应具备过充保护、过流保护、短路保护

    2.5 型式认可要求¶

    电池系统型式认可流程：

    申请 → 技术文件审查 → 样品试验 → 工厂审查 → 认可证书

    主要试验项目：

    试验项目 标准 说明
    外观检查 — 结构、标识、防护等级
    充放电性能 IEC 62619 容量、倍率、效率
    过充试验 IEC 62619 不起火不爆炸
    过放试验 IEC 62619 不泄漏不起火
    短路试验 IEC 62619 不起火不爆炸
    热冲击 IEC 62619 高低温循环
    挤压试验 IEC 62619 不起火不爆炸
    热失控试验 CCS 规范 单体触发热失控，不蔓延
    振动试验 IEC 60068 船用振动环境
    盐雾试验 IEC 60068 海上盐雾腐蚀
    EMC 试验 IEC 60945 电磁兼容

    3. 解读笔记¶

    3.1 "A-60 级防火分隔"¶

    原文： 电池舱与客舱、机舱之间应采用 A-60 级防火分隔。

    解读： - "A-60"指 A 级分隔（钢质+绝缘），60 分钟内背火面温升不超过 139°C - 实际设计中常用：岩棉板 50mm + 钢板，或陶瓷纤维毯 + 钢板 - 需要注意穿舱件（电缆管）的防火密封处理

    3.2 "热失控不蔓延"¶

    原文： 电池系统应进行热失控试验，单体热失控不应蔓延至相邻单体。

    解读： - 试验方法：强制触发电芯热失控（加热或针刺） - 判定标准：相邻电芯最高温度 < 100°C，无明火、无爆炸 - 设计对策：模组间加隔热板（气凝胶/云母板）、合理间距 - 这是最严苛的试验项目，很多电池系统在此项被卡
